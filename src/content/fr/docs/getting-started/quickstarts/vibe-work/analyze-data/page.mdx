# Analyser un jeu de données

Téléchargez un tableur dans <AppLink href="https://chat.mistral.ai/work" app="vibe">Work</AppLink> et posez vos questions en langage naturel. Work choisit les outils appropriés (lecteur de fichiers, sandbox de code intégrée, Canvas) pour calculer des synthèses, générer des graphiques et réaliser des analyses complémentaires, sans que vous ayez besoin d’écrire du code.

**Temps requis :** ~10 minutes

<SectionTab as="h2" sectionId="prerequisites">Prérequis</SectionTab>

- Un compte Mistral (le plan Free suffit pour un essai, un plan payant est conseillé pour traiter des datasets volumineux).
- Un fichier de données à analyser (CSV, XLSX ou JSON).

<SectionTab as="h2" sectionId="step-1">Étape 1 : Télécharger vos données</SectionTab>

1. Ouvrez <AppLink href="https://chat.mistral.ai/work" app="vibe">chat.mistral.ai</AppLink> et passez sur l’onglet **Work**.
2. Dans la barre d’outils de la conversation, cliquez sur l’icône `+`, puis sélectionnez `Upload Files` et choisissez votre fichier de données (par exemple `sales-q4-2025.csv`).
3. Work affiche un aperçu du fichier. Vérifiez que l’aperçu correspond à vos attentes.

Pour de meilleurs résultats, privilégiez des fichiers avec des en-têtes de colonnes explicites. Work les analyse pour comprendre le schéma avant toute opération.





<SectionTab as="h2" sectionId="step-2">Étape 2 : Poser une question sur vos données</SectionTab>

Saisissez une question en langage naturel. Work écrit et exécute du code dans une sandbox, puis affiche des tableaux, des graphiques et des explications directement dans **[Canvas](/vibe/work/files-and-canvas)**, un panneau latéral qui s’ouvre automatiquement dès que Work doit présenter un document, un graphique ou une sortie longue modifiable. Vous pouvez aussi le demander explicitement, par exemple : *« ouvrir dans Canvas »* ou *« visualiser ce fichier »*.

**Exemples :**

> Résume ce jeu de données. Combien de lignes et de colonnes le composent ? Quelles sont les statistiques principales par colonne ?

> Affiche-moi l’évolution mensuelle du chiffre d’affaires sous forme de courbe.

> Quels sont les cinq produits les plus vendus ? Montre-les sur un graphique en barres.

> Calcule la corrélation entre dépenses marketing et chiffre d’affaires, puis explique le résultat.

Work affiche le code exécuté à côté du résultat, ce qui vous permet de vérifier la logique appliquée.

<SectionTab as="h2" sectionId="step-3">Étape 3 : Affiner et exporter</SectionTab>

Faites évoluer l’analyse en posant des questions complémentaires. Work conserve les données en Contexte tout au long de la conversation.

1. **Modifier un graphique** : *« Élargis le graphique et ajoute les étiquettes de données. »*
2. **Filtrer les données** : *« Affiche uniquement les lignes où la région est Europe. »*
3. **Comparer des périodes** : *« Compare le chiffre d’affaires T3 ou T4 par catégorie de produits. »*

Pour enregistrer votre travail, utilisez le bouton **Exporter** dans Canvas pour télécharger les tableaux au format CSV/Excel ou les graphiques en image. Vous pouvez aussi copier des tableaux directement depuis Canvas, ou demander à Work de *« ouvrir dans Canvas »* pour transformer l’analyse en document éditable.

<SectionTab as="h2" sectionId="verify">Vérification</SectionTab>

Votre analyse fonctionne si :

- Work identifie correctement les noms de colonnes et les types de données.
- Les graphiques générés correspondent aux données du fichier.
- Les questions complémentaires utilisent le même dataset sans retéléchargement.
- Les résultats statistiques (moyenne, médiane, corrélation) sont conformes à vos attentes.

Si une colonne est mal interprétée, précisez-le explicitement : *« La colonne “Date” est au format JJ/MM/AAAA. »*
