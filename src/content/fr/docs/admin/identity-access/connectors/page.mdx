# Connecteurs

Les administrateurs d'organisation utilisent les Connecteurs dans Admin pour connecter des applications partagées, consulter les détails des Connecteurs et contrôler quels outils Connecteur sont disponibles pour les utilisateurs dans Vibe et Studio.

L'administration des Connecteurs s'applique aux niveaux Organisation et espace de travail. Utilisez Admin pour gérer les accès. Utilisez [les Connecteurs de Studio](/studio-api/connectors) lorsque vous avez besoin de la documentation Connecteur pour les développeurs, d'exemples d'API ou du débugueur de Connecteur.

<SectionTab as="h1" sectionId="open-connectors">Ouvrir les Connecteurs dans Admin</SectionTab>

1. Ouvrez <AppLink href="https://admin.mistral.ai/organization/connectors" path={["Panneau Admin", "Administration", "Connecteurs"]} />.
2. Consultez les sections **Connexions aux applications** et **Tous les Connecteurs**.

<SectionTab as="h1" sectionId="app-connections">Connecter des applications au niveau organisation</SectionTab>

Utilisez **Connexions aux applications** pour connecter des services une fois au niveau Organisation. Une fois connectés, les membres peuvent utiliser l'application via Mistral sans se connecter individuellement.

| Application | Ce qu'elle fait |
| --- | --- |
| **GitHub** | Recherchez des dépôts, consultez les issues et gérez les pull requests dans GitHub. |
| **Slack** | Recherchez des messages, lisez des canaux, envoyez des messages et gérez des canvas dans Slack. |

Pour connecter une application, cliquez sur **Connecter à GitHub** ou **Connecter à Slack**, puis complétez le processus d'autorisation.

<SectionTab as="h1" sectionId="all-connectors">Gérer tous les Connecteurs</SectionTab>

**Tous les Connecteurs** liste les Connecteurs et outils pouvant être utilisés dans Vibe et Studio. Ouvrez un Connecteur pour consulter sa configuration et contrôler l'accès aux outils.

Chaque page Connecteur comprend trois onglets :

- **Aperçu** : consultez les métadonnées, la visibilité, l'authentification et le contenu disponible.
- **Identifiants** : consultez ou ajoutez des identifiants pour le Connecteur.
- **Permissions** : choisissez quels outils peuvent être utilisés par le modèle.

<SectionTab as="h2" variant="secondary" sectionId="overview-tab">Consulter les détails du Connecteur</SectionTab>

L'onglet **Aperçu** affiche les informations de base sur le Connecteur, notamment :

| Champ | Exemple |
| --- | --- |
| **Visibilité** | Organisation |
| **Authentification** | OAuth2 |
| **Origine** | Mis en avant |
| **Capacités** | Lecture seule |
| **Nom du Connecteur** | atlassian |
| **ID du Connecteur** | 0198e70f-57b0-77f6-a752-0a7f5ea2da35 |

Si vous n'êtes pas connecté, la section **Inclut** affiche **Vous n'êtes pas connecté**. Cliquez sur **Connecter** pour vous authentifier et voir les outils.

<SectionTab as="h2" variant="secondary" sectionId="credentials-tab">Gérer les identifiants</SectionTab>

L'onglet **Identifiants** liste les identifiants disponibles pour le Connecteur. S'il n'y a pas encore d'identifiants, cliquez sur **Connecter** pour commencer à utiliser le Connecteur.

:::tip
Utilisez les connexions aux applications au niveau organisation pour un accès bot partagé. Utilisez les identifiants Connecteur lorsque les utilisateurs ou les espaces de travail ont besoin d'une connexion à un compte spécifique.
:::

<SectionTab as="h2" variant="secondary" sectionId="permissions-tab">Définir les permissions du Connecteur</SectionTab>

L'onglet **Permissions** contrôle quels outils peuvent être utilisés par le modèle.

| Permission | Effet |
| --- | --- |
| **Autorisé** | Tous les outils sont disponibles pour les utilisateurs dans cet espace de travail. |
| **Restreint** | Seuls les outils sélectionnés sont disponibles pour les utilisateurs dans cet espace de travail. |
| **Bloqué** | Le Connecteur est désactivé pour les utilisateurs dans cet espace de travail. |

Utilisez **Restreint** lorsqu'un Connecteur inclut des outils qui ne doivent pas être disponibles pour chaque workflow ou conversation. Utilisez **Bloqué** lorsqu'un Connecteur ne doit pas être utilisé dans un espace de travail.
