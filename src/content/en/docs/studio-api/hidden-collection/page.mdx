# Hidden Collection

This hidden collection gives us a safe place to test direct URL access, sidebar filtering, search filtering, and sitemap filtering.

## What to test

Use this collection to confirm these behaviors:

- The collection doesn't appear in the sidebar.
- The collection doesn't appear in search results.
- The collection doesn't appear in the sitemap.
- Each page stays available from its direct URL.
