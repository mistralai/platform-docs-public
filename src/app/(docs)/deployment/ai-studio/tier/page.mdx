# Rate limits & usage tiers

Our API enforces rate limits to ensure fair usage, balance load, and prevent abuse.

:::note
Check the [rate limits page](https://admin.mistral.ai/plateforme/limits) for current values and usage tiers for your Workspace.
:::

<SectionTab as="h1" sectionId="how-do-the-rate-limits-work">How rate limits work</SectionTab>

We limit how much a Workspace can use the API through two types of rate limits:

- **Requests per second (RPS)**
- **Tokens per minute/month**

Rate limits are set at the **Workspace level** and determined by your **usage tier**. Each tier defines its own set of limits. If you need higher limits, contact us via the support button with details about your use case.

<SectionTab as="h1" sectionId="usage-tiers">Usage tiers</SectionTab>

View the rate and usage limits for your Workspace in the AI Studio [limits](https://admin.mistral.ai/plateforme/limits) section.

We offer several tiers, including a **free API tier** with restrictive rate limits for exploring the API. For production use, upgrade to a higher tier.
