# Guide de démarrage

Créez un pipeline RAG en 5 minutes : ingérez des documents dans un vecteur store, puis effectuez des recherches dessus.

<SectionTab as="h1" sectionId="prerequisites">Prérequis</SectionTab>

- **Python 3.12+**
- **Docker** (pour exécuter Vespa en local)
- Une **clé API Mistral** à récupérer sur [console.mistral.ai](https://console.mistral.ai/)

<SectionTab as="h1" sectionId="install">Installation</SectionTab>

Installez Search Toolkit avec le plugin Vespa :

```bash
pip install "mistralai-search-toolkit[vespa]"
```

Ou avec [uv](https://docs.astral.sh/uv/) :

```bash
uv add "mistralai-search-toolkit[vespa]"
```

<SectionTab as="h1" sectionId="setup-vespa">Configurer Vespa</SectionTab>

Lancez une instance locale de Vespa avec Docker :

```bash
docker run --detach \
  --name vespa \
  --hostname vespa-container \
  --publish 8080:8080 \
  --publish 19071:19071 \
  vespaengine/vespa
```

Attendez que Vespa soit prêt :

```bash
curl --retry 10 --retry-delay 3 --retry-all-errors \
  http://localhost:19071/state/v1/health
```

Définissez votre clé API Mistral :

```bash
export MISTRAL_API_KEY=your-api-key
```

<SectionTab as="h1" sectionId="ingest">Ingestion de documents</SectionTab>

Créez un pipeline qui charge les fichiers, extrait le texte, segmente en morceaux, génère les embeddings, puis indexe dans Vespa :

```python
import asyncio
from pathlib import Path

from mistralai.client import Mistral
from mistralai.search.toolkit.embedders import MistralEmbedder
from mistralai.search.toolkit.ingestion.extractors import PlainTextExtractor
from mistralai.search.toolkit.ingestion.loaders import FilesystemFileLoader
from mistralai.search.toolkit.ingestion.pipelines import Pipeline
from mistralai.search.toolkit.ingestion.text_splitters import CharacterTextSplitter
from mistralai.search.toolkit.plugins.vespa import VespaClient, VespaConfig


async def main():
    mistral_client = Mistral(api_key="your-api-key")

    # Configuration de Vespa
    vector_store = VespaClient(
        config=VespaConfig(
            collection_name="quickstart_collection",
            query_endpoint="http://localhost:8080",
            feed_endpoint="http://localhost:8080",
            embedding_model="mistral-embed",
        ),
        query_profile=None,  # Non requis pour l’ingestion
    )

    # Création du pipeline
    pipeline = Pipeline(
        loader=FilesystemFileLoader(),
        extractor=PlainTextExtractor(),
        text_splitter=CharacterTextSplitter(chunk_size=500),
        embedder=MistralEmbedder(client=mistral_client),
        vector_store=vector_store,
    )

    # Création de l’index et ingestion
    await vector_store.create_index()
    await pipeline.run(
        documents=[Path("doc1.txt"), Path("doc2.txt")],
        collection_name="quickstart_collection",
    )
    print("Documents ingérés !")


asyncio.run(main())
```

Le pipeline enchaîne cinq étapes :

1. **`FilesystemFileLoader`** lit les fichiers bruts depuis le disque.
2. **`PlainTextExtractor`** extrait le texte du fichier. Pour les PDF, utilisez plutôt `MistralOCRExtractor`.
3. **`CharacterTextSplitter`** découpe le texte en segments de 500 caractères.
4. **`MistralEmbedder`** génère un embedding vectoriel pour chaque segment.
5. **`VespaClient`** indexe chaque segment, stocke à la fois l’embedding (recherche vectorielle) et le texte brut (pour la recherche par mots-clés).

<SectionTab as="h1" sectionId="search">Recherche</SectionTab>

Interrogez les documents indexés avec la recherche vectorielle :

```python
import asyncio

from mistralai.client import Mistral
from mistralai.search.toolkit.embedders import MistralEmbedder
from mistralai.search.toolkit.plugins.vespa import VespaClient, VespaConfig
from mistralai.search.toolkit.plugins.vespa.search import (
    VespaQueryProfile,
    VespaQueryProfileConfig,
)
from mistralai.search.toolkit.retrieval import QueryEngine
from mistralai.search.toolkit.retrieval.retrievers import VectorRetriever


async def main():
    mistral_client = Mistral(api_key="your-api-key")
    embedder = MistralEmbedder(client=mistral_client)

    # Configurez Vespa avec un profil de requête spécifique à la recherche
    query_profile = VespaQueryProfile(
        config=VespaQueryProfileConfig(name="default", version=None),
        namespace="quickstart_collection",
    )
    vector_store = VespaClient(
        config=VespaConfig(
            collection_name="quickstart_collection",
            query_endpoint="http://localhost:8080",
            feed_endpoint="http://localhost:8080",
            embedding_model="mistral-embed",
        ),
        query_profile=query_profile,
    )

    # Création du moteur de requête
    query_engine = QueryEngine(
        retriever=[VectorRetriever(client=vector_store, embedder=embedder)],
    )

    # Recherche
    result = await query_engine.search(
        query="What is RAG?",
        top_k=5,
        include_metadata=True,
        include_content=True,
    )

    for i, r in enumerate(result.results, 1):
        print(f"{i}. [Score: {r.score:.3f}] {r.chunk.content[:200]}...")


asyncio.run(main())
```

<SectionTab as="h1" sectionId="pdf-ingestion">Ingestion de PDF avec OCR</SectionTab>

Pour les documents PDF, remplacez `PlainTextExtractor` par `MistralOCRExtractor` et utilisez `MarkdownTextSplitter` pour une découpe tenant compte de la structure :

```python
from mistralai.search.toolkit.ingestion.extractors import MistralOCRExtractor
from mistralai.search.toolkit.ingestion.text_splitters import (
    MarkdownTextSplitter,
    MarkdownTextSplitterConfig,
)

pipeline = Pipeline(
    loader=FilesystemFileLoader(),
    extractor=MistralOCRExtractor(client=mistral_client),
    text_splitter=MarkdownTextSplitter(
        MarkdownTextSplitterConfig(chunk_size=5048, chunk_overlap=50)
    ),
    embedder=MistralEmbedder(client=mistral_client),
    vector_store=vector_store,
)
```
