# Comment contribuer

Merci de votre intérêt pour contribuer à Mistral AI. Nous accueillons toutes les personnes qui souhaitent contribuer et nous apprécions votre temps et vos efforts !

## Rejoignez notre programme d'ambassadeurs

:::tip[ ]
Rendez-vous sur [Programme d'ambassadeurs](../ambassador/) pour en savoir plus et postuler !
:::

## Contribuer à la documentation officielle

Si vous souhaitez contribuer à notre [documentation officielle](https://docs.mistral.ai), veuillez soumettre une PR sur [https://github.com/mistralai/platform-docs-public](https://github.com/mistralai/platform-docs-public).

Vous pouvez facilement aider en :

- corrigeant une faute de frappe
- clarifiant une section
- documentant une fonctionnalité sous-documentée
- mettant à jour une section qui aurait dû l'être
- ...

## Contribuer aux clients de code

### Client Python

Soumettez une PR pour améliorer notre client Python : [https://github.com/mistralai/client-python](https://github.com/mistralai/client-python).

Le moyen le plus simple de commencer est de consulter la [liste des issues](https://github.com/mistralai/client-python/issues) et de voir laquelle pourrait vous intéresser.

### Client JavaScript

Soumettez une PR pour améliorer notre client JavaScript : [https://github.com/mistralai/client-js](https://github.com/mistralai/client-js)

Le moyen le plus simple de commencer est de consulter la [liste des issues](https://github.com/mistralai/client-js/issues) et de voir laquelle pourrait vous intéresser.

## Contribuer à la communauté

### Mistral Cookbook

Si vous avez des exemples intéressants mettant en valeur les modèles Mistral, que vous utilisiez l'API Mistral ou les modèles open-weight, n'hésitez pas à les partager en soumettant une PR sur notre [dépôt cookbook](https://github.com/mistralai/cookbook).

### Discord

Nous sommes reconnaissants envers toute personne qui partage des ressources et nous aide à répondre aux questions sur notre communauté [Discord](https://discord.gg/mistralai).

### Rayonnement communautaire

Une manière précieuse de soutenir Mistral AI est de participer activement à la communication dans la communauté. Cela peut inclure des activités comme le partage sur les réseaux sociaux, la rédaction d'articles de blog, la création de vidéos tutorielles démontrant votre travail avec l'API ou les modèles Mistral AI. N'hésitez pas à nous taguer sur les réseaux sociaux, nous serons là pour vous soutenir !
