# Agents

Transform Mistral models from text generators into active problem-solvers. The Agents API combines powerful language models with built-in tools for code execution, web search, image generation, and external integrations, enabling multi-step reasoning and autonomous task completion.

## In this section

- [**Introduction**](/studio-api/agents/introduction): Get started with agents, understand core concepts, and learn how agents differ from standard chat completions.

- [**Agents API**](/studio-api/agents/agents-api): Create, configure, and manage agents programmatically. Handle conversations, streaming, branching, and agent lifecycle operations.

- [**Agent Tools**](/studio-api/agents/agent-tools): Equip agents with capabilities: built-in tools (web search, code interpreter, image generation, document libraries), custom function calling, and the function calling API for structured tool use.

- [**Handoffs**](/studio-api/agents/handoffs): Orchestrate multiple specialized agents to collaborate on complex tasks, passing control between them based on the problem at hand.
