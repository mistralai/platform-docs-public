# Installation

Mistral Vibe is a Python package you can **install and use directly in your terminal**.

<SectionTab as="h1" sectionId="install-script">Install script</SectionTab>

The fastest way to get started is to **run the install script** for your platform:

<Tabs groupId="os">
    <TabItem value="mac-linux" label="Mac/Linux" default>
```bash
curl -LsSf https://mistral.ai/vibe/install.sh | bash
```
    </TabItem>
    <TabItem value="windows" label="Windows">
```bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex" && uv tool install mistral-vibe
```
    </TabItem>
</Tabs>

:::warning
Vibe works on Windows, but we officially support and target UNIX environments.
:::

<SectionTab as="h1" sectionId="manual">Manual installation</SectionTab>

**Prerequisite:** Python 3.12 or later.

Install `mistral-vibe` with your preferred package manager:

<Tabs>
    <TabItem value="uv" label="uv" default>
```bash
uv tool install mistral-vibe
```
    </TabItem>
    <TabItem value="pip" label="pip">
```bash
pip install mistral-vibe
```
    </TabItem>
</Tabs>

:::tip
If you run into issues with `uv`, try the `--no-cache` flag.
:::

Once installed, head to the [quickstart](quickstart) to start using Vibe.

<SectionTab as="h1" sectionId="repo">Browse the source code</SectionTab>

Mistral Vibe is released under the [Apache License 2.0](https://github.com/mistralai/mistral-vibe/blob/main/LICENSE) and the source code is available on [GitHub](https://github.com/mistralai/mistral-vibe).

Clone the repository to install locally or customize the code to your needs:

```bash
git clone https://github.com/mistralai/mistral-vibe.git
cd mistral-vibe
```
