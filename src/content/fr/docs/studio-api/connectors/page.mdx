# Connecteurs

:::info
Les Connecteurs sont une fonctionnalité **Public Preview**. L’interface de l’API peut évoluer.
:::

Les Connecteurs sont des serveurs [MCP](https://modelcontextprotocol.io/) enregistrés que vous pouvez utiliser **comme outils dans les conversations et les Agents**, depuis n’importe quel SDK ou directement via l’API, **sans gérer le transport MCP localement**.

Une fois enregistrés, un Connecteur **expose ses outils au modèle** à la demande : le modèle les découvre automatiquement et appelle le bon outil **en fonction de la requête de l’utilisateur**.

Les clés API peuvent être limitées à l’accès aux Connecteurs. Lors de la création d’une clé API, choisissez si elle peut appeler uniquement les Connecteurs partagés avec votre espace de travail ou à la fois vos Connecteurs privés et ceux partagés avec l’espace de travail. Voir [Clés API](/admin/security-access/api-keys#connector-access-scope) pour les options d’accès.

:::tip[Qu’est-ce que le MCP ?]
Le [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) est une norme ouverte pour connecter les modèles d’IA à des outils et sources de données externes via une interface unifiée. Tout **serveur compatible MCP** peut être enregistré comme un Connecteur.
:::
