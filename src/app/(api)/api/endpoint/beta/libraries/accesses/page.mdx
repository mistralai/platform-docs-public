## <Pill variant="primary"><b>PUT</b></Pill> /v1/libraries/\{library\_id\}/share \{#operation-libraries_share_create_v1\}

</OperationTitleSection>

<OperationSummarySection slot="summary">

_Create or update an access level._

</OperationSummarySection>

<OperationDescriptionSection slot="description">

Given a library id, you can create or update the access level of an entity. You have to be owner of the library to share a library. An owner cannot change their own role. A library cannot be shared outside of the organization.

</OperationDescriptionSection>
