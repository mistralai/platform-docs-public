# Créer votre organisation

Lorsque vous vous inscrivez sur Mistral, une Organisation est créée automatiquement. Si vous devez créer une Organisation supplémentaire : par exemple, pour séparer un compte personnel d’un compte professionnel : suivez les étapes ci-dessous.

<SectionTab as="h1" sectionId="create">Créer une organisation</SectionTab>

1. Cliquez sur le nom de votre organisation dans le coin supérieur gauche de Studio.
2. Cliquez sur **Ajouter** dans la section Organisations.
3. Saisissez un nom pour l’Organisation.
4. Acceptez les conditions générales d’utilisation.
5. Cliquez sur **Créer une organisation**.

La nouvelle Organisation commence avec vous comme seul Admin et un espace de travail par défaut.

<SectionTab as="h1" sectionId="view-organization-details">Consulter les détails de l’Organisation</SectionTab>

Une fois votre Organisation créée, utilisez la page **Votre Organisation** pour consulter le nom de l’Organisation, son ID et le nombre de membres actifs.

1. Ouvrez <AppLink href="https://admin.mistral.ai/organization" app="admin">Organisation</AppLink> dans l’Admin Console.
2. Consultez **Membres actifs** pour voir combien de membres actifs appartiennent à l’Organisation.
3. Consultez ou mettez à jour le **nom de l’Organisation**.
4. Copiez l’**ID de l’Organisation** lorsque vous en avez besoin pour le support ou vos enregistrements internes.

<SectionTab as="h1" sectionId="delete-organization">Supprimer une Organisation</SectionTab>

La section **Danger zone** des paramètres de l’Organisation permet de supprimer définitivement une Organisation.

:::danger
La suppression d’une Organisation est définitive. Elle supprime l’Organisation et ses ressources, données, espaces de travail, clés API, paramètres, historique d’utilisation et autres contenus rattachés à l’Organisation. Elle ne supprime pas les comptes utilisateurs qui étaient membres de l’Organisation.
:::

1. Ouvrez <AppLink href="https://admin.mistral.ai/organization" path={["Admin Console", "Administration", "Organization"]} />.
2. Accédez à la section **Danger zone**.
3. Cliquez sur **Delete Organization**.
4. Confirmez la suppression.

<SectionTab as="h1" sectionId="next-steps">Prochaines étapes</SectionTab>

Après avoir créé votre Organisation :

- [Envoyer des invitations](/admin/identity-access/user-management) : ajoutez des utilisateurs et attribuez des rôles.
- [Vérifier votre domaine](/admin/set-up-organization/verify-domain) : nécessaire pour activer l’authentification par domaine d’e-mail ou le SSO.
- [Configurer la facturation](/admin/billing-usage/billing) : ajoutez un moyen de paiement pour activer l’accès à l’API au-delà de la version gratuite.
