## Introduction

Le fine-tuning est une technique puissante pour personnaliser et optimiser les performances des grands modèles de langage (LLM) pour des cas d'usage spécifiques. En entraînant davantage un LLM pré-entraîné sur un jeu de données labellisé lié à une tâche particulière, le fine-tuning peut améliorer les performances du modèle. Cela peut se faire avec un grand modèle pour des tâches complexes ou dissemblables, ou avec un modèle plus petit pour égaler les performances d'un modèle plus grand, offrant potentiellement des avantages en termes de latence et de coût. L'augmentation des performances varie selon les cas d'usage.

Mistral AI propose une API de fine-tuning via [La Plateforme](https://console.mistral.ai/), facilitant le fine-tuning de tous nos modèles open-source et commerciaux. Pour ceux qui souhaitent expérimenter avec différents paramètres et acquérir une compréhension approfondie du processus de fine-tuning, nous proposons également l'accès à notre base de code open-source [mistral-finetune](https://github.com/mistralai/mistral-finetune/). Cependant, nous recommandons d'utiliser notre API pour tous les fine-tunings car elle ne nécessite pas de GPU, propose diverses options et permet également le fine-tuning de nos modèles commerciaux.

Dans ce guide, nous aborderons les sujets suivants :

- Préparer le jeu de données
- Exemples de bout en bout avec l'API Mistral
- Exemples de bout en bout avec `mistral-finetune`
