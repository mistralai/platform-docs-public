## <Pill variant="primary"><b>GET</b></Pill> /v1/workflows/\{workflow\_name\}/metrics \{#operation-get_workflow_metrics_v1_workflows_workflow_name_metrics_get\}

</OperationTitleSection>

<OperationSummarySection slot="summary">

_Obtenir les métriques d'un workflow_

</OperationSummarySection>

<OperationDescriptionSection slot="description">

Obtenir des métriques complètes pour un workflow spécifique.

Args:
    workflow_name: Le nom du type de workflow pour lequel obtenir les métriques
    start_time: Filtre de temps de début optionnel (format ISO 8601)
    end_time: Filtre de temps de fin optionnel (format ISO 8601)

Retours:
    WorkflowMetrics: Dictionnaire contenant les métriques:
        - execution_count: Nombre total d'exécutions
        - success_count: Nombre d'exécutions réussies
        - error_count: Nombre d'exécutions échouées/terminées
        - average_latency_ms: Durée moyenne d'exécution en millisecondes
        - retry_rate: Proportion de workflows avec des nouvelles tentatives
        - latency_over_time: Données chronologiques des durées d'exécution

Exemple:
    GET /v1/workflows/MyWorkflow/metrics
    GET /v1/workflows/MyWorkflow/metrics?start_time=2025-01-01T00:00:00Z
    GET /v1/workflows/MyWorkflow/metrics?start_time=2025-01-01T00:00:00Z&end_time=2025-12-31T23:59:59Z

</OperationDescriptionSection>
