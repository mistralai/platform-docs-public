# Limites d'utilisation

Définissez des plafonds de dépenses et des seuils d'utilisation pour contrôler les coûts dans les espaces de travail de votre Organisation.

<SectionTab as="h1" sectionId="workspace-limits">Limites de dépenses par espace de travail</SectionTab>

Définissez un plafond de dépenses mensuel par espace de travail pour éviter les frais imprévus. Lorsqu'un espace de travail atteint sa limite, les requêtes API sont rejetées jusqu'à la prochaine période de facturation ou jusqu'à ce que vous augmentiez la limite.

Pour configurer une limite de dépenses :

1. Ouvrez les <AppLink href="https://admin.mistral.ai" app="admin">paramètres de l'espace de travail</AppLink>.
2. Définissez la **limite de dépenses mensuelle**.
3. Enregistrez vos modifications.

<SectionTab as="h1" sectionId="rate-limits">Limites de débit</SectionTab>

Les limites de débit s'appliquent au niveau de l'espace de travail et varient selon le [palier d'utilisation](/admin/user-management-finops/tier). Elles comprennent :

- **Requêtes par seconde (RPS)** : nombre maximal de requêtes API simultanées.
- **Tokens par minute** : limite de débit pour le traitement des tokens.
- **Tokens par mois** : plafond de consommation global.

Consultez vos limites de débit actuelles dans <AppLink href="https://admin.mistral.ai/plateforme/limits" app="admin">Limites</AppLink>.

<SectionTab as="h1" sectionId="monitoring">Surveiller l'utilisation</SectionTab>

Suivez la consommation d'API, l'utilisation des tokens et les coûts par espace de travail depuis le panneau d'administration. Depuis les paramètres de l'espace de travail, cliquez sur l'onglet **Utilisation** pour voir :

- Un aperçu de vos dépenses
- Une répartition détaillée par API et services
- Les coûts en tokens d'entrée et de sortie par modèle

Utilisez ces données avec les limites de dépenses pour maîtriser les coûts dans vos équipes et projets.
