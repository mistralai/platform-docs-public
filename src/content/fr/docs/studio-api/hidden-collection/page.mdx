# Collection masquée

Cette collection masquée fournit un espace de test pour vérifier l’accès par URL directe, le filtrage dans la barre latérale, le filtrage dans la recherche et le filtrage dans le plan du site.

## À tester

Utilisez cette collection pour confirmer ces comportements :

- La collection n’apparaît pas dans la barre latérale.
- La collection n’apparaît pas dans les résultats de recherche.
- La collection n’apparaît pas dans le plan du site.
- Chaque page reste accessible depuis son URL directe.
