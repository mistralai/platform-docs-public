# Mistral Vibe

<Image
  url='/img/vibe-logo.svg'
  alt="Mistral Vibe logo"
  width="200px"
  centered
/>

Mistral Vibe is a **command-line coding assistant** that lets you explore, modify, and interact with your codebase **using natural language**.

Describe what you want in plain English (refactor a module, find a bug, scaffold a feature) and Vibe handles the rest: reading files, running commands, and writing code **directly in your project**.

It's open source ([GitHub](https://github.com/mistralai/mistral-vibe), Apache 2.0) and works with any Mistral model or your own local setup.

<SectionTab as="h1" sectionId="objects">What Vibe can do</SectionTab>

Vibe covers the full development loop, from **understanding your codebase** to **writing and testing changes**:

- **Interactive chat**: A conversational agent that understands your requests, breaks down complex tasks, and works through them step by step.
- **Built-in tools**: Read, write, and patch files. Run shell commands. Search code with grep. Track tasks with a built-in todo list... all without leaving the terminal!
- **Project-aware context**: Vibe reads your project structure and Git status automatically, so it knows where you are and what you're working on.
- **CLI ergonomics**: Autocompletion, persistent history, and customizable themes.
- **Configurable**: Swap models, providers, tool permissions, and UI preferences in a single configuration file.
- **Agents & skills**: Create multiple agents with different capabilities and permissions, and compose them into workflows.
- **Controlled execution**: You stay in control. Every tool call goes through an approval step.
