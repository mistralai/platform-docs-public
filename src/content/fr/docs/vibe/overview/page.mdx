# Vibe

:::info
**Le Chat s'appelle désormais Vibe.** L’URL <AppLink href="https://chat.mistral.ai">chat.mistral.ai</AppLink> reste le point d’accès principal ; la majorité des fonctionnalités se trouvent désormais dans **Work**, avec quelques fonctionnalités héritées encore disponibles dans **Chat** (voir ci-dessous). Votre offre, votre historique et vos paramètres sont toujours présents dans **Chat**.
:::

Vibe est l’**agent unifié** de Mistral pour la productivité et le développement, conçu pour un usage professionnel et accessible sur le web, mobile, dans votre éditeur de code ou le terminal.

Il fonctionne selon trois modes, chacun adapté à un type de tâche spécifique :

- **Work** : tâches professionnelles complexes, sur plusieurs étapes, orchestrées entre vos applications et outils (recherche, brouillons, synthèses, exécutions programmées).
- **Code** : développement dans le terminal, dans VS Code ou en session cloud à distance.
- **Chat** : conversations rapides par échanges, avec les fonctions héritées du Chat (Agents, mode "Réflexion", Recherche, interpréteur de code, Souvenirs).

:::tip
Besoin d’aide pour choisir le mode adapté ? Comparez-les dans [Choisir Chat, Work ou Code](/vibe/choose-chat-work-code).
:::

<SectionTab as="h2" sectionId="how-vibe-works">Comment fonctionne Vibe</SectionTab>

Quel que soit le mode, vous interagissez avec Vibe **de la même manière** : mêmes modèles Mistral, mêmes logiques, même contrôle. Un workflow Vibe typique se déroule ainsi :

1. **Vous exprimez votre besoin** en langage naturel.
2. **Vibe rassemble le contexte** à partir de votre demande, des fichiers joints, des outils connectés ou du web.
3. **Vibe planifie et exécute** : il découpe la tâche, lance les bons outils, affiche sa progression en temps réel.
4. **Vous révisez et réutilisez** : les résultats sont inspectables, modifiables et réutilisables en session.

**À chaque étape, vous gardez la main.** Vibe demande votre accord avant chaque action sensible, affiche tous les appels d’outils et vous laisse piloter ou interrompre quand vous le souhaitez.
