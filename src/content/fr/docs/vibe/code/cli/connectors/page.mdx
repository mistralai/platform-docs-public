# Connecteurs

Les connecteurs permettent à la CLI Vibe Code d'accéder à **des services tiers ou Mistral pris en charge** directement depuis une session de code. Contrairement aux [serveurs MCP](/vibe/code/cli/mcp-servers), que vous configurez vous-même, les connecteurs sont **des intégrations gérées par Mistral**.

:::info
Pour la liste complète des connecteurs disponibles et leurs fonctionnalités, consultez [Connecteurs dans Vibe Work](/vibe/work/connectors). Vibe Code et Vibe Work partagent ces mêmes intégrations.
:::

<SectionTab as="h2" sectionId="vs-mcp">Connecteurs vs serveurs MCP</SectionTab>

Les deux exposent des outils à Vibe, mais leur mode de gestion diffère. Choisissez celui adapté au service que vous souhaitez connecter :

| | Connecteurs | Serveurs MCP |
|---|---|---|
| **Configuré par** | Mistral, en tant qu'intégrations | Vous, dans `config.toml` |
| **Mode d'authentification** | Connexion au compte ou au service | En-têtes, variables d'environnement ou clés API |
| **Découvrabilité** | Sélectionnés, listés dans la CLI | Tout service compatible MCP |
| **Idéal pour** | Services courants avec intégrations gérées | Outils personnalisés, serveurs privés, outils expérimentaux |

<SectionTab as="h2" sectionId="manage">Gérer les connecteurs</SectionTab>

La détection et la configuration des connecteurs sont automatiques. Vous pouvez les lister, activer, désactiver et inspecter depuis une session CLI.

Lister tous les connecteurs et serveurs MCP :

```text
/connectors
```

(`/connectors` est un alias de `/mcp`.)

Dans la liste, avec un connecteur sélectionné :

| Touche | Action |
|---|---|
| `E` | Activer le connecteur |
| `D` | Désactiver le connecteur |

Indiquez un nom pour afficher les outils exposés par un connecteur précis :

```text
/connectors fetch_server
```

<SectionTab as="h2" sectionId="stay-in-control">Gardez le contrôle</SectionTab>

- **Vérifiez ce qu’un connecteur peut accéder** avant de l’authentifier.
- **Privilégiez les droits en lecture seule ou restreints** lorsque l’action ne nécessite pas d’écriture.
- **Révoquez les connecteurs inutilisés** depuis le service concerné lorsque vous n’en avez plus besoin.
