# Que sont les agents IA ?

Les agents IA sont des systèmes autonomes propulsés par de grands modèles de langage (LLM) qui, à partir d'instructions de haut niveau, peuvent planifier, utiliser des outils, exécuter des étapes de traitement et agir pour atteindre des objectifs spécifiques. Ces agents utilisent des capacités avancées de traitement du langage naturel pour comprendre et exécuter des tâches complexes, et peuvent collaborer entre eux pour atteindre des résultats plus sophistiqués.

  width="425px"
  centered
/>

Notre API Agents and Conversations permet aux développeurs de créer de tels agents, en exploitant de nombreuses fonctionnalités telles que :
- Plusieurs modèles multimodaux disponibles, **modèles texte et vision**.
- **État persistant** à travers les conversations.
- Possibilité de converser avec des **modèles de base**, **un seul agent** et **plusieurs agents**.
- Outils connecteurs intégrés pour l'**exécution de code**, la **recherche web**, la **génération d'images** et la **bibliothèque de documents** prêts à l'emploi.
- **Capacité de transfert** pour utiliser différents agents dans un workflow, permettant aux agents d'appeler d'autres agents.
- Les fonctionnalités supportées via notre endpoint de chat completions sont également prises en charge, notamment :
  - **Structured Outputs**
  - **Document Understanding**
  - **Tool Usage**
  - **Citations**
