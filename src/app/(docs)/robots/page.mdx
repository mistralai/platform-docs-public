## Mistral AI crawlers

We employ web crawlers, aka "robots", and user agents to execute tasks for our products, either automatically or upon user request. To help webmasters manage how their sites and content interact with AI, we use specific robots.txt tags.

<SectionTab as="h1" sectionId="mistralai-user">MistralAI-User</SectionTab>

`MistralAI-User` is for **user actions in le Chat**. When users ask le Chat a question, it may **visit a web page** to help answer and **include a link to the source in its response**.

`MistralAI-User` governs which sites these user requests can be made to. It's **not used for crawling the web in any automatic fashion, nor to crawl content for generative AI training**.

For more information about Mistral AI's policy on web crawlers used for generative AI training, see the [Mistral AI Legal Center](https://legal.mistral.ai/).
