# Projects

Projects let you **group conversations** into distinct, manageable collections. If you're juggling client work, internal research, and a product launch at the same time, Projects keep each thread of work organized and easy to find.

:::info
Projects are available on all plans. Limits may apply to the number of Projects and chats per Project. See our [pricing page](https://mistral.ai/pricing) for details.
:::

<SectionTab as="h1" sectionId="creating-a-project">Creating a Project</SectionTab>

1. Click the `New Project` button next to the `Projects` header in the left sidebar.
2. Choose a name and an icon for your Project.
3. Click `Create`.

Your new Project appears in the sidebar, ready to receive conversations.

<SectionTab as="h1" sectionId="project-instructions">Project instructions and context</SectionTab>

Each Project can have its own instructions and shared context, similar to [custom instructions](/le-chat/knowledge-integrations/custom-instructions) but scoped to the Project.

- **Instructions**: set a tone, define formatting rules, or provide domain-specific guidance. These apply to every conversation within the Project.
- **Additional context**: upload files that are shared across all chats in the Project. Every conversation can reference these files without you having to re-upload them.
- **Cross-chat context**: include other chats from the Project as context, so Le Chat can draw on previous conversations when answering.

:::note
Adding a Library to a Project isn't supported yet. You can still attach Libraries to individual chats within the Project.
:::

<SectionTab as="h1" sectionId="managing-chats">Managing chats</SectionTab>

The Project overview page shows all your conversations in list or grid view. From there you can add, move, or remove chats.

### Adding a chat

**Drag and drop**

1. Locate the chat in the sidebar.
2. Drag it over the target Project name and release.

**Folder icon**

1. Open the chat.
2. Click the folder icon in the chat header.
3. Select a Project from the dropdown.

### Moving a chat to another Project

Open the chat, click the folder icon, and select a different Project. The chat moves there automatically.

### Removing a chat

**From the Project overview**

1. Open the Project.
2. Click the three-dots menu on the chat card.
3. Select `Remove from project`.

**From the chat itself**

1. Open the chat.
2. Click the folder icon.
3. Deselect the current Project.

Removing a chat from a Project doesn't delete it. The chat stays in your main chat history and can be reassigned at any time.

:::note
A chat can only belong to one Project at a time.
:::

<SectionTab as="h1" sectionId="deleting-a-project">Deleting a Project</SectionTab>

1. Open the Project.
2. Click the three-dots menu in the top-right corner.
3. Select `Delete` and confirm.

:::caution
Deleting a Project is permanent and can't be undone. The chats inside aren't deleted. They return to your main chat history and can be reassigned to another Project.
:::
