- Points de terminaison API : nous avons renommé 3 points de terminaison API et ajouté 2 points de terminaison de modèle. <Tag api/>

  - `open-mistral-7b` (alias `mistral-tiny-2312`) : renommé depuis `mistral-tiny`. Le point de terminaison `mistral-tiny` sera obsolète dans trois mois.
  - `open-mixtral-8x7B` (alias `mistral-small-2312`) : renommé depuis `mistral-small`. Le point de terminaison `mistral-small` sera obsolète dans trois mois.
  - `mistral-small-latest` (alias `mistral-small-2402`) : nouveau modèle. <Tag model/>
  - `mistral-medium-latest` (alias `mistral-medium-2312`) : ancien modèle. L’ancien `mistral-medium` a été daté et étiqueté comme `mistral-medium-2312`. Le point de terminaison `mistral-medium` sera obsolète dans trois mois.
  - `mistral-large-latest` (alias `mistral-large-2402`) : notre nouveau modèle phare à la performance de pointe. <Tag model/>

- Nouvelles capacités de l’API : <Tag api/>

  - [Function calling](/studio-api/conversations/function-calling) : disponible pour Mistral Small et Mistral Large.
  - [JSON mode](/studio-api/conversations/structured-output#json-mode) : disponible pour Mistral Small et Mistral Large

- [La Plateforme](https://console.mistral.ai/) : <Tag other/>

  - Nous avons ajouté la gestion de plusieurs devises pour le système de paiement, y compris la possibilité de payer en dollars américains.
  - Nous avons introduit des fonctionnalités pour les entreprises, notamment la gestion des administrateurs, qui permet aux utilisateurs de gérer les membres de leur organisation.

- [Le Chat](https://chat.mistral.ai/) : <Tag other />
  - Nous avons lancé la toute nouvelle interface conversationnelle Le Chat pour interagir facilement avec les modèles Mistral.
  - Vous pouvez actuellement interagir avec trois modèles : Mistral Large, Mistral Next et Mistral Small.
