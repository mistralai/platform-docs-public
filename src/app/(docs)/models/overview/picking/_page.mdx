This guide helps you choose the right Mistral model for your use case. Each model targets a different balance of capability, speed, and cost.

## Choose by use case

| Use case | Recommended model | Why |
|----------|-------------------|-----|
| **General-purpose chat, reasoning, coding** | Mistral Large 3 | Most capable generalist. Best accuracy on complex tasks. |
| **Balanced quality and cost** | Mistral Medium 3.1 | Frontier-class multimodal at a lower price than Large. |
| **High-volume, low-latency tasks** | Mistral Small 3.2 | Fast and affordable. Good for classification, extraction, summarization. |
| **Edge and on-device** | Ministral 3 (3B / 8B / 14B) | Small footprint. Available in 3B, 8B, and 14B parameter variants. |
| **Complex reasoning (math, logic, proofs)** | Magistral Medium 1.2 | Reasoning model with extended thinking. Best for multi-step problems. |
| **Fast reasoning** | Magistral Small 1.2 | Efficient reasoning model for lighter reasoning tasks. |
| **Code generation and completion** | Codestral | Optimized for code. Supports fill-in-the-middle and code completion. |
| **Agentic coding (SWE tasks)** | Devstral 2 | Frontier agent for autonomous software engineering workflows. |
| **Document processing and OCR** | OCR 3 | Extract text, tables, and structure from PDFs and images. |
| **Audio transcription** | Voxtral Mini Transcribe 2 | Transcribe audio files with high accuracy. |
| **Live audio transcription** | Voxtral Mini Transcribe Realtime | Stream-based transcription for real-time applications. |
| **Content moderation** | Mistral Moderation | Classify text across 9 policy categories. |
| **Code embeddings** | Codestral Embed | Semantic representations optimized for code search and similarity. |

## Decision framework

Start with these questions:

1. **Do you need vision or multimodal input?**
   - Yes: Mistral Large 3, Mistral Medium 3.1, or Ministral 3 (all support vision)
   - Audio input: Voxtral Mini Transcribe 2 or Voxtral Small

2. **Is deep reasoning required?**
   - Yes: Magistral Medium 1.2 (or Magistral Small 1.2 for faster, cheaper reasoning)
   - No: continue to step 3

3. **What is your quality vs. cost priority?**
   - Quality first: Mistral Large 3
   - Balanced: Mistral Medium 3.1
   - Cost first: Mistral Small 3.2 or Ministral 3

4. **Is it a specialized task?**
   - Code: Codestral or Devstral 2
   - Documents: OCR 3
   - Audio: Voxtral Mini Transcribe 2
   - Moderation: Mistral Moderation

## Model tiers

| Tier | Description | Models |
|------|-------------|--------|
| **Premier** | Hosted exclusively on Mistral's API. Best performance. | Mistral Large 3, Mistral Medium 3.1, Magistral Medium 1.2 |
| **Open** | Weights available for self-hosting and fine-tuning. | Mistral Small 3.2, Ministral 3, Codestral, Devstral 2, Magistral Small 1.2 |

## Performance vs. cost tradeoff

Start with **Mistral Large 3** when accuracy matters most. Evaluate whether the results meet your requirements, then try **Mistral Medium 3.1** or **Mistral Small 3.2** to reduce costs. If the smaller model produces acceptable quality for your task, use it.

For high-volume batch workloads, use the [Batch API](/studio-api/batch-processing) to reduce costs by up to 50%.
