# Chat

The chat interface is where you interact with Le Chat, your AI assistant. Type a message, press `Enter ↵`, and Le Chat responds immediately.

:::info
When Le Chat generates a response, it draws on **context** from multiple sources:
- Your current message
- The full conversation history
- Enabled tools (e.g., [web search](/le-chat/research-analysis/web-search), [Code Interpreter](/le-chat/content-creation/code-interpreter))
- [Uploaded files](/le-chat/research-analysis/files-upload)
- Active [Connectors](/le-chat/knowledge-integrations/connectors)
- [Library](/le-chat/knowledge-integrations/libraries) content
- [Memories](/le-chat/knowledge-integrations/memories)
- Your [custom instructions](/le-chat/knowledge-integrations/custom-instructions)

The richer the context, the more accurate and tailored the response.
:::

This page covers the basics: sending messages, working in multiple languages, and sharing conversations.

<SectionTab as="h1" sectionId="sending-messages">Sending messages</SectionTab>

Type your message in the chat box and press `Enter ↵` to send. Your message is sent instantly, and the assistant responds in real time.

Le Chat keeps context across messages within a conversation, so you can build on previous exchanges without repeating yourself.

:::tip
Click the `+` icon or type `/` to enable capabilities for the current conversation.
:::

<SectionTab as="h1" sectionId="use-cases">Common use cases</SectionTab>

- **Draft and review documents**: write reports, proposals, or client emails. Review existing documents for tone, clarity, and consistency.
- **Translate business content**: translate contracts, correspondence, or internal documentation between languages. Le Chat detects the source language automatically.
- **Analyze data and run calculations**: work through formulas, financial models, or data questions. For verified accuracy, use [Code Interpreter](/le-chat/content-creation/code-interpreter) to run the computation.
- **Write and debug code**: generate new code, troubleshoot existing snippets, or get a code review with best-practice suggestions.
- **Research and summarize**: get explanations, competitive analysis, or quick summaries on any topic.

:::tip
Be specific about the format and depth you need. A detailed prompt *"Summarize this contract in three bullet points, in French"* gives better results than a vague one. For best practices, see our [prompting guide](/models/best-practices/prompt-engineering).
:::

<SectionTab as="h1" sectionId="multilingual-support">Multilingual support</SectionTab>

Le Chat detects the language of your message and responds in that same language. You don't need to specify a language unless you want the response in a different one.

You can change the interface language in your account settings. This doesn't affect the language the model uses in responses, which is always driven by your prompt.

<SectionTab as="h1" sectionId="sharing-conversations">Sharing conversations</SectionTab>

You can share any conversation with others:

1. Click the `Share` icon button in the top-right corner of the conversation.
2. Choose a method:
   - **Direct share**: share to X, LinkedIn, Reddit, WhatsApp, or email.
   - **Share by link**: generate a URL you can copy and send.
3. Shared conversations display a green dot indicator next to the `Share` button.

**How sharing works:**

- Conversations are **private by default**. Others only see them when you explicitly share.
- Sharing creates a **read-only copy** of the conversation. The recipient sees it as it was at the time of sharing. Messages you add afterward won't appear in the shared version.

:::warning
Shared links **can be revoked** using the `Link accessible to anyone` toggle. Anyone with an active link can access the content, so review the conversation for sensitive information before sharing.
:::
