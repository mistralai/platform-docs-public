# RAG & Embeddings

Build custom retrieval-augmented generation pipelines with embeddings, vector search, and model generation. These guides show how retrieval works under the hood and how to build a RAG pipeline from scratch.

If you want a managed feature that ingests, vectorizes, and searches documents for you, use [Libraries](/studio-api/libraries). If you want to ground responses in connected sources such as Google Drive or SharePoint, use [Connectors](/studio-api/connectors).
