Lors de la diffusion en flux, vous disposerez d'indices spécifiques pour les différents types de contenu. Ceux-ci incluent :
- `conversation.response.started` : Début de la réponse de la conversation.
- `conversation.response.done` : La réponse est terminée.
- `conversation.response.error` : Une erreur s'est produite.
- `message.output.delta` : Fragment de contenu, généralement des tokens correspondant à la réponse du modèle.
- `tool.execution.started` : L'exécution d'un outil a démarré.
- `tool.execution.done` : L'outil a terminé son exécution.
- `agent.handoff.started` : Le transfert vers un autre agent a démarré.
- `agent.handoff.done` : Le transfert est terminé.
- `function.call.delta` : Fragment de contenu, généralement des tokens correspondant à l'appel de l'outil fonction.
