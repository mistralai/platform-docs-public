# Automated provisioning (SCIM)

System for Cross-domain Identity Management (SCIM) lets an identity provider create, update, and deactivate users automatically. Use it when your Organization needs automated provisioning and deprovisioning for compliance or large-scale user lifecycle management.

:::note
SCIM documentation is out of scope for the first Admin documentation release. We are waiting for confirmation from the IAM team on availability, setup steps, and group sync behavior.
:::

<SectionTab as="h1" sectionId="expected-scope">Expected scope</SectionTab>

Expected coverage includes:

- SCIM availability and prerequisites;
- where to create the SCIM token;
- setup with supported identity providers;
- user and group sync behavior;
- deprovisioning behavior when users are removed or disabled in the identity provider.
