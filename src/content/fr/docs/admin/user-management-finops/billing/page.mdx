# Facturation

Consultez et gérez les moyens de paiement, les factures et les informations de facturation de votre organisation.

Nécessite le rôle **Admin** ou **facturation**.

<SectionTab as="h1" sectionId="setup">Configurer la facturation</SectionTab>

1. Ouvrez <AppLink href="https://admin.mistral.ai/organization/billing" app="admin">Abonnements › Facturation</AppLink>.
2. Ajoutez un mode de paiement (carte bancaire ou Google Pay).
3. Renseignez les informations de facturation : nom de la personne ou de l’entreprise, adresse et numéro de TVA le cas échéant.

La configuration de la facturation est nécessaire pour :

- Activer des clés API au-delà du niveau gratuit.
- Souscrire aux offres Pro, Team ou Enterprise.

<SectionTab as="h1" sectionId="invoices">Factures</SectionTab>

Consultez et téléchargez vos factures dans la section facturation du panneau Admin. Les factures incluent :

- Les frais d’abonnement
- Les frais d’utilisation API (paiement à l’usage)
- Les allocations de sièges

<SectionTab as="h1" sectionId="payment-schedule">Calendrier de paiement</SectionTab>

- **Abonnements Vibe** (Pro, Team, Enterprise) : facturés au début de chaque période de facturation (mensuelle ou annuelle). Il s’agit d’offres à prix fixe pour l’accès à Vibe.
- **Utilisation API** (offre Scale) : facturée selon la consommation (jetons traités). Aucun coût initial ; vous ne payez que ce que vous consommez.
- **Montées en gamme** : en passant à une offre Vibe supérieure, le règlement du nouveau plan est immédiat.

<SectionTab as="h1" sectionId="billing-scope">Périmètre de facturation</SectionTab>

La facturation est gérée au niveau de l’organisation. Toute consommation d’API dans un espace de travail de votre organisation est regroupée sur une seule facture. Utilisez les [plafonds de consommation](/admin/user-management-finops/usage-limits) pour fixer des limites de dépenses par espace de travail.
