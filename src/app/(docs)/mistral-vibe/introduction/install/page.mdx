# Installation

Mistral Vibe is a Python package that can be installed and used directly in your terminal as a command-line tool. We provide below a quick installation guide.

Perfect for:
- Developers working **terminal-first**.
- Anyone who wants an AI assistant that interacts with their **local environment**.
- Teams needing reproducible, script-friendly workflows in the **command line**.

<SectionTab as="h1" sectionId="install">Install</SectionTab>

You can quickly install `mistral-vibe` by running the following command in your terminal:

<Tabs groupId="os">
    <TabItem value="mac-linux" label="Mac/Linux" default>
```bash
curl -LsSf https://mistral.ai/vibe/install.sh | bash
```
    </TabItem>
    <TabItem value="windows" label="Windows">

```bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex" && uv tool install mistral-vibe
```
    </TabItem>
</Tabs>

You can also install via `uv` directly or via `pip` with the following:

<Tabs>
    <TabItem value="uv" label="UV" default>
```
uv tool install mistral-vibe
```
    </TabItem>
    <TabItem value="pip" label="Pip">
```bash
pip install mistral-vibe
```
    </TabItem>
</Tabs>

Be sure to have installed Python 3.12+, if you encounter issues with `uv`, try using the `--no-cache` flag.

:::warning
Mistral Vibe works on Windows, but we officially support and target UNIX environments.
:::

Once installed, you are ready to get started, visit the [Quickstart](quickstart) guide to learn how to use Mistral Vibe.

<SectionTab as="h1" sectionId="repo">Repository</SectionTab>

The source code for Mistral Vibe is available on [GitHub](https://github.com/mistralai/mistral-vibe). You can clone the repository and install it locally, or customize the code to suit your needs.

```bash
git clone https://github.com/mistralai/mistral-vibe.git
cd mistral-vibe
```

Feel free to contribute, visit [CONTRIBUTING.md](https://github.com/mistralai/mistral-vibe/blob/main/CONTRIBUTING.md) to learn how.
