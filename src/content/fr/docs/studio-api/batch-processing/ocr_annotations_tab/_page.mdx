<SectionTab as="h2" variant="secondary" sectionId="ocr-explained">Explications</SectionTab>

```py
{
    "custom_id": "2", # Un identifiant de métadonnée qui sera retourné dans le fichier de résultat pour retrouver la requête
    "body": { # Corps de la requête
        "document": { # Document à traiter
            "image_url": "data:image/jpeg;base64,<base64_image>" # Ici, on utilise une image encodée en base64, mais vous pouvez aussi utiliser une URL de fichier, et document_url pour un PDF ou autre type de document
        },
        "document_annotation_format": {...}, # Schéma d’annotation de document, voir la documentation Annotations
        "bbox_annotation_format": {...}, # Schéma d’annotation bbox, voir la documentation Annotations
        "pages": [0,1,2,3,4,5,6,7] # Pages à annoter, limitation à 8 pages si vous utilisez l’annotation de document
    }
}
```
