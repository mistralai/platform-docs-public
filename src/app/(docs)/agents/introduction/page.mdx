# What are AI Agents?

An AI Agent is a **specialized assistant** you configure with instructions, tools, and knowledge. Give it a task in natural language, and it plans the steps, calls the right tools, and works through them on your behalf.

You can create Agents visually in Mistral AI Studio's [Playground](https://console.mistral.ai/build/playground) or programmatically through our [Agents and Conversations API](../agents/agents).

Here's what makes them powerful:

- **Multimodal models**: text and vision, including [Mistral Large 3](../models/mistral-large-3-25-12) and [Mistral Medium 3.1](../models/mistral-medium-3-1-25-08).
- **Built-in tools**: code execution, web search, image generation, and document library.
- **Persistent state**: conversations carry context across turns.
- **Handoffs**: Agents can delegate tasks to each other within a workflow.
- **Multiple setups**: base models, single-Agent, or multi-Agent configurations.
- **Chat Completions features**: Structured Outputs, Document Understanding, Tool Usage, and Citations.

<SectionTab as="h1" sectionId="get-started">Get started</SectionTab>

Learn the API, then add tools and workflows:

1. [Agents & conversations](../agents/agents): Create your first Agent and manage conversations.
2. [Tools](../agents/tools): Give your Agents capabilities.
   - [Built-in tools](../agents/tools/built-in): Web search, Code Interpreter, image generation, and Document Library.
   - [Function calling](../agents/tools/function_calling): Create custom tools.
   - [MCP](../agents/tools/mcp): Connect third-party tools via MCP servers.
3. [Handoffs](../agents/handoffs): Delegate tasks between Agents in multi-Agent workflows.
