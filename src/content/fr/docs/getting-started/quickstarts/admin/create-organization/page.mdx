# Créer votre organisation

Configurez une organisation Mistral à partir d'un nouveau compte via <AppLink href="https://admin.mistral.ai">Admin</AppLink>.

- Activez la facturation pour que les clés API fonctionnent
- Créez votre premier espace de travail
- Invitez des membres de l'équipe et attribuez des rôles

Une fois terminé, votre équipe dispose de clés API fonctionnelles et d'un accès à la plateforme.

**Temps nécessaire :** ~15 minutes

<SectionTab as="h2" sectionId="prerequisites">Prérequis</SectionTab>

- Un compte Mistral AI. <AppLink href="https://console.mistral.ai">Créer un compte</AppLink>
- Une carte bancaire ou un compte de facturation cloud (AWS ou Azure Marketplace).
- L'adresse e-mail d'au moins un collègue à inviter.

<SectionTab as="h2" sectionId="step-1">Étape 1 : Créer votre organisation</SectionTab>

Connectez-vous à <AppLink href="https://console.mistral.ai">Studio</AppLink>. Les nouveaux comptes sont invités à nommer leur organisation et sont redirigés automatiquement vers Admin.

Vous avez déjà un compte sans organisation ? Ouvrez <AppLink href="https://admin.mistral.ai">Admin</AppLink> et suivez les instructions de configuration.

<SectionTab as="h2" sectionId="step-2">Étape 2 : Activer la facturation</SectionTab>

Les membres de l'équipe ne peuvent pas générer de clés API fonctionnelles tant que la facturation n'est pas active.

Ouvrez <AppLink href="https://admin.mistral.ai/organization/billing" app="admin">Abonnements › Facturation</AppLink>, cliquez sur **Ajouter un moyen de paiement** et connectez votre carte ou votre compte marketplace cloud. Définissez éventuellement une **Limite de dépenses mensuelles** avant d'enregistrer.

L'activation de la facturation prend 2 à 3 minutes. Si des membres de l'équipe signalent des clés API inactives juste après, demandez-leur de patienter et de réessayer.

<SectionTab as="h2" sectionId="step-3">Étape 3 : Créer un espace de travail</SectionTab>

Votre espace de travail par défaut est créé automatiquement : ignorez cette étape si un seul espace de travail suffit.

Ouvrez <AppLink href="https://admin.mistral.ai/organization/workspaces" app="admin">Gestion › Espaces de travail</AppLink>, cliquez sur **Créer un espace de travail** et donnez-lui un nom (par exemple, « Engineering » ou « Production »).

<SectionTab as="h2" sectionId="step-4">Étape 4 : Inviter des membres de l'équipe</SectionTab>

Ouvrez <AppLink href="https://admin.mistral.ai/organization/members" app="admin">Administration › Membres</AppLink>, cliquez sur **Inviter un membre**, saisissez l'adresse e-mail et attribuez un rôle :

<SectionTab as="h2" sectionId="step-5">Étape 5 : Sécurité et supervision</SectionTab>

À mesure que votre équipe grandit, contrôlez qui peut accéder à quoi.

**Configurer le SSO :**
Imposez l'authentification d'entreprise avec SSO basé sur SAML. Accédez à **Administration › Paramètres** dans le panneau Admin pour configurer votre fournisseur d'identité et définir un **Domaine d'e-mail autorisé** pour bloquer les invitations vers des adresses personnelles ou externes.

**Superviser les journaux d'audit :**
Ouvrez <AppLink href="https://admin.mistral.ai/audit-logs" app="admin">Administration › Journaux d'audit</AppLink> pour suivre les événements critiques. Vous disposez ainsi d'un registre indiquant quand les clés API ont été créées, qui a été invité et quand les paramètres de facturation ont changé.

<SectionTab as="h2" sectionId="verify">Vérifier</SectionTab>

Le membre de l'équipe invité reçoit un e-mail en quelques minutes. Une fois qu'il accepte, son statut dans <AppLink href="https://admin.mistral.ai/organization/members" app="admin">Administration › Membres</AppLink> passe de **En attente** à **Actif**.
