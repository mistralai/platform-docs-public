# Travailler avec les fichiers et Canvas

Les **fichiers** servent à importer la matière première dans une tâche. **Canvas** est l’éditeur intégré dans lequel Work produit, prévisualise et affine les résultats.

Ensemble, ils vous permettent de passer d’un simple téléchargement (CSV, PDF, capture d’écran, document) à un livrable éditable et abouti, sans quitter l’interface de chat.

:::info
Les fichiers et Canvas sont disponibles sur **tous les plans** sans configuration requise. Les limites d’usage dépendent de votre plan. Consultez notre [page tarifs](https://mistral.ai/pricing) pour plus de détails.
:::

<SectionTab as="h2" sectionId="files">Fichiers : importez des sources dans votre tâche</SectionTab>

Téléchargez directement des documents, images ou tableurs dans le chat. Work lit le contenu et l’utilise comme contexte : vous pouvez poser des questions, extraire des informations, et obtenir des réponses basées sur vos propres données.

<SectionTab as="h3" variant="secondary" sectionId="how-to-upload">Comment importer</SectionTab>

Deux options pour ajouter des fichiers dans une discussion :

- **Glisser-déposer** : faites glisser un ou plusieurs fichiers dans la fenêtre de chat.
- **Bouton `+`** : cliquez sur l’icône `+` dans la barre d’outils du chat, puis sélectionnez `Importer des fichiers` et choisissez-les sur votre appareil.

Vous pouvez importer plusieurs fichiers à la fois (voire un dossier complet). Work les garde en contexte tout au long de la discussion : posez toutes vos questions de suivi sans avoir à les télécharger de nouveau.

:::tip
Utilisez des noms de fichiers explicites : cela facilite la référence à un fichier précis quand il y en a plusieurs dans le même chat.
:::

<SectionTab as="h3" variant="secondary" sectionId="supported-formats">Formats pris en charge</SectionTab>

| Catégorie | Formats |
|----------|---------|
| **Documents** | PDF, Word (`.docx`, `.doc`), PowerPoint (`.pptx`, `.ppt`), ODT, EPUB, RTF |
| **Tableurs** | Excel (`.xlsx`, `.xls`), CSV, ODS, Numbers |
| **Images** | PNG, JPEG, WebP, GIF |
| **Texte et balisage** | TXT, Markdown, RST, LaTeX |
| **Formats de données** | JSON, JSONL, XML, YAML |
| **Fichiers de code** | Python, JavaScript/TypeScript, Java, Go, Rust, C/C++, Ruby, PHP, SQL et plus |
| **E-mail** | EML, MSG |

<SectionTab as="h3" variant="secondary" sectionId="querying-documents">Interroger des documents texte</SectionTab>

Importez un PDF, un fichier Word ou une présentation et demandez à Work de cibler les informations qui vous intéressent. Les réponses proviennent directement de votre contenu importé, jamais de connaissances générales.

Exemples de requêtes :

- *« Résume ce rapport annuel en insistant sur les risques financiers. »*
- *« Extrait toutes les dates, parties et obligations présentes dans ce contrat. »*
- *« Liste les principaux constats de sécurité dans cet audit. »*
- *« Compare les diapositives 3 et 7 et liste les différences. »*

Poursuivez ensuite : Work garde les fichiers en mémoire pendant toute la discussion, pour affiner, approfondir ou demander des traductions, sans avoir à tout réimporter.

<SectionTab as="h3" variant="secondary" sectionId="analyzing-images">Analyser des images</SectionTab>

Work sait interpréter photos, schémas, captures d’écran et pages numérisées. Importez une image via le bouton `+` et posez votre question.

- **Extraction de texte** : *« Extrait tout le texte de cette facture scannée. »* Fonctionne aussi avec des notes manuscrites, des formulaires imprimés ou un tableau blanc en photo.
- **Interprétation de schéma** : *« Décris l’architecture visible sur ce schéma. »*
- **Analyse de capture d’écran** : *« Quelle erreur s’affiche sur cette capture d’écran ? »*

:::note
Pour des besoins d’analyse documentaire volumineuse (extraction structurée depuis PDF scannés, formulaires multipages, lots de manuscrits) : explorez les fonctions [Document AI](/studio-api/document-processing/overview) de Studio.
:::

<SectionTab as="h3" variant="secondary" sectionId="analyzing-spreadsheets">Analyser des tableurs</SectionTab>

Pour un fichier CSV ou Excel, Work peut répondre à vos questions, générer des synthèses ou extraire des données directement depuis l’import.

Exemples de requêtes :

- *« Montre-moi toutes les ventes de la région Nord. »*
- *« Quel est le montant moyen par affaire, trimestre par trimestre ? »*
- *« Liste les 10 clients principaux par chiffre d’affaires. »*

:::note
Pour de l’analyse de données Python (statistiques avancées, graphiques, transformations), utilisez le [Code Interpreter](/vibe/chat-legacy/code-interpreter) dans Chat.
:::

<SectionTab as="h2" sectionId="canvas">Canvas : relisez, éditez et itérez sur vos résultats</SectionTab>

Canvas est l’éditeur intégré dans lequel Work produit et affine du texte, des données, du code et des présentations. Passez d’un fichier téléchargé brut à un livrable prêt à l’emploi sans quitter la discussion.

<SectionTab as="h3" variant="secondary" sectionId="activation">Activation</SectionTab>

1. Cliquez sur l’icône `+` ou tapez `/` dans la fenêtre de chat.
2. Sélectionnez `Outils` puis activez `Canvas`.

Une fois activé, Canvas **s’ouvre automatiquement** si Work le juge pertinent. Vous pouvez aussi **le déclencher vous-même** en tapant `/canvas` dans le champ de saisie, ou en demandant à Work d’*« ouvrir dans Canvas »* dans votre prompt.

Exemples de requêtes ouvrant Canvas :

- *« Crée un plan de présentation pour nos résultats T3. »*
- *« Analyse le CSV joint et affiche-le sous forme de tableau. »*
- *« Écris un script Python pour analyser des logs JSON. »*

<SectionTab as="h3" variant="secondary" sectionId="canvas-features">Fonctionnalités de Canvas</SectionTab>

<ExplorerTabs>
<ExplorerTab value="editing" label="Édition" default>

Travaillez directement dans Canvas comme dans tout éditeur.

- **Édition directe** : cliquez partout pour modifier texte, code ou données à la main. Corrigez une faute, renommez une colonne ou réécrivez un paragraphe.
- **Plusieurs Canvas** : ouvrez plusieurs Canvas dans une discussion. Chacun apparaît sous forme d’onglet : passez d’un tableau de données à un brouillon de rapport sans perdre le contexte.
- **Historique des versions** : utilisez les flèches pour naviguer entre versions. Activez la vue suivi des modifications pour les visualiser. Restaurez à tout moment une version précédente.

</ExplorerTab>
<ExplorerTab value="ai-assistance" label="Aide par l’IA">

Laissez Work reformuler, traduire, ou transformer votre contenu.

- **Prompts intégrés** : sélectionnez un passage pour faire apparaître le champ `Demander à Work`. Tapez une demande (*« rends ce passage plus concis »*, *« traduis en français »*), Work applique la modification uniquement à la sélection.
- **Actions rapides** : cliquez sur l’icône étoile pour accéder à des raccourcis contextuels (traduire, optimiser, visualiser…). Les actions proposées varient selon le type de contenu.
- **Mode aperçu** : cliquez sur l’icône œil pour basculer entre le contenu brut et le rendu final. Prend en charge HTML, composants React, [diagrammes Mermaid](https://mermaid.js.org/) (diagrammes/textes, graphes, organigrammes) et [présentations Marp](https://marp.app/) (slides textuelles).

</ExplorerTab>
<ExplorerTab value="export" label="Export et partage">

Récupérez le contenu produit sur Canvas pour l’utiliser dans vos propres outils.

- **Exporter** : sauvegardez le rendu Canvas sous forme de fichier. Par exemple, exportez des slides en PowerPoint ou un tableau en fichier CSV/Excel.
- **Partager** : diffusez toute la discussion, contenu Canvas inclus.
- **Copier** : transférez le contenu brut de Canvas dans votre presse-papiers.

</ExplorerTab>
</ExplorerTabs>

<SectionTab as="h2" sectionId="use-cases">Cas d’usage courants</SectionTab>

- **Rapports trimestriels** : importez vos données financières, générez tableaux et synthèses, exportez au format slides.
- **Brief projet** : rédigez des documents structurés avec édition assistée par l’IA, en ligne.
- **Revue de contrat** : extrayez clauses, parties, dates et obligations depuis un PDF.
- **Exploration de données** : interrogez CSV et tableurs avec édition directe.
- **Analyse d’image et capture d’écran** : extrait du texte, interprète des schémas, identifie des erreurs.
- **Présentations** : construisez et prévisualisez des slides Marp sans quitter Work.
- **Prototypage code** : rédigez et testez des scripts Python ou des composants React.
- **Emails et brouillons de contenu** : rédigez, corrigez, traduisez du texte via des prompts intégrés.

<SectionTab as="h2" sectionId="practical-workflow">Cas pratique : de vos données à des slides</SectionTab>

Fichiers et Canvas prennent tout leur intérêt combinés. Exemple classique : transformer une source brute en livrable prêt à partager.

<SectionTab as="h3" variant="secondary" sectionId="upload-explore">Importez et explorez vos données</SectionTab>

Importez un CSV ou un tableur avec le bouton `+` puis `Importer des fichiers`, et demandez :

```text
Analyse le CSV joint et ouvre-le dans Canvas.
```

Canvas affiche vos données sous forme de tableau modifiable. Cliquez dans une cellule pour corriger des en-têtes, renommer des colonnes ou retirer des lignes vides.

:::tip
Si Canvas s’ouvre dans un mauvais format, demandez *« Affiche ça sous forme de tableau »* pour changer d’affichage.
:::

<SectionTab as="h3" variant="secondary" sectionId="generate-summary">Générez un résumé écrit</SectionTab>

Une fois vos données en place, demandez à Canvas de produire une synthèse écrite :

```text
Transforme ce tableau en résumé exécutif concis.
```

Canvas propose alors un texte basé sur vos données. Vous pouvez éditer le résultat ou utiliser des prompts intégrés pour affiner certains passages.

<SectionTab as="h3" variant="secondary" sectionId="create-slides">Créez des slides</SectionTab>

Transformez votre analyse en présentation prête à diffuser :

```text
Produis des points clés prêts à être intégrés en slides, avec recommandations.
```

Canvas génère des slides au format [Marp](https://marp.app/) (présentations textuelles). Cliquez sur l’icône **œil** pour prévisualiser le rendu.

Quand vos slides sont prêtes, exportez-les en PowerPoint pour vos ajustements finaux.

<SectionTab as="h2" sectionId="tips">Conseils pour de meilleurs résultats</SectionTab>

- **Soyez précis.** Indiquez votre objectif : *« Résume ce contrat pour un responsable conformité »* donnera de meilleurs résultats que *« Résume ce fichier. »*
- **Utilisez des verbes d’action.** Résumer, extraire, comparer, expliquer, valoriser, lister, convertir…
- **Référencez vos fichiers.** Si vous avez plusieurs imports, nommez-les : *« Dans `AnnualReport_2024.pdf`, page 14, extraire le détail du chiffre d’affaires. »*
- **Itérez.** Affinez, traduisez, ou reformatez en poursuivant l’échange.
- **Vérifiez avant de partager.** Considérez chaque résultat comme un brouillon. Vérifiez faits, chiffres et sources à partir de l’original.
