# Langues prises en charge

Les modèles de langage de Mistral sont formés sur des données multilingues et prennent en charge un large éventail de langues. Les listes ci-dessous reflètent les langues pour lesquelles on peut s’attendre à des **performances élevées**. IFNEn raison de la manière dont les modèles sont formés, ils peuvent également offrir de bonnes performances dans d’autres langues non répertoriées ici.

<SectionTab as="h1" sectionId="language-models">Modèles de langage</SectionTab>

| Région | Langues |
|--------|-----------|
| **Europe** | français, allemand, espagnol, portugais, italien, néerlandais, polonais, tchèque, danois, finnois, grec, norvégien, roumain, suédois, croate, serbe, ukrainien, catalan, breton |
| **Moyen-Orient** | arabe, persan, hébreu, turc, ourdou |
| **Asie du Sud** | hindi, bengali, gujarati, kannada, marathi, népalais, pendjabi, tamoul, telugu |
| **Asie du Sud-Est** | indonésien, laotien, malais, tagalog, thaï, vietnamien |
| **Asie de l’Est** | chinois, japonais, coréen, russe |
