- We released OCR 4 (`mistral-ocr-4-0`). `mistral-ocr-latest` now points to it. <Tag model/>
  - Introducing `include_blocks` in our OCR API. When set to `true`, each page returns a `blocks` array with paragraph-level bounding boxes and a structural label (`text`, `title`, `list`, `table`, `image`, `equation`, `caption`, `code`, `references`, `aside_text`, `header`, `footer`, `signature`) in reading order. Learn more in our [OCR documentation](/studio-api/document-processing/basic_ocr#block-extraction). <Tag api/>
  - The `pages` parameter in our OCR API now also accepts a string of comma-separated digits and ranges (e.g. `"0,1,2"`, `"0-5"`, or `"0,2-4"`) in addition to a list of integers. <Tag api/>
