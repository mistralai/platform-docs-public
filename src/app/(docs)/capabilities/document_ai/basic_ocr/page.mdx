<SectionTab as="h1" sectionId="before-you-start">Before you start</SectionTab>

### Key features

- **Text extraction** that preserves document structure, headers, paragraphs, lists, and tables.
- **Table formatting** via the `table_format` parameter:
  - `null`: tables are returned inline as markdown within the page.
  - `markdown`: tables are returned as separate markdown tables.
  - `html`: tables are returned as separate HTML tables.
- **Header and footer extraction** via `extract_header` and `extract_footer`. When enabled, headers and footers are returned in dedicated fields instead of the main content.
- Results in **markdown format** for easy parsing and rendering.
- Handles **complex layouts** including multi-column text, mixed content, and hyperlinks.
- Supports multiple document formats:
  - `image_url`: PNG, JPEG/JPG, AVIF, and more.
  - `document_url`: PDF, PPTX, DOCX, and more.
  - See the [FAQ](#faq) for a comprehensive list.

Learn more in the [OCR endpoint documentation](https://docs.mistral.ai/api/endpoint/ocr).

:::info
Table formatting and header/footer extraction are available for OCR 2512 and newer versions.
:::

The OCR processor returns extracted **text content**, **image bounding boxes**, and document structure metadata.

The output is a JSON object containing extracted text, image bounding boxes, metadata, and document structure:

```py
{
  "pages": [ # Content of each page
    {
      "index": int, # Page index
      "markdown": str, # Raw markdown content
      "images": list, # Image information when images are extracted
      "tables": list, # Table information when using table_format=html or table_format=markdown
      "hyperlinks": list, # Detected hyperlinks
      "header": str|null, # Header content when using extract_header=True
      "footer": str|null, # Footer content when using extract_footer=True
      "dimensions": dict # Page dimensions
    }
  ],
  "model": str, # Model used for OCR
  "document_annotation": dict|null, # Annotation information (see Annotations docs)
  "usage_info": dict # Usage information
}
```

:::note
Extracted images and tables are replaced with placeholders in the markdown:
- `![img-0.jpeg](img-0.jpeg)`
- `[tbl-3.html](tbl-3.html)`

Map them to actual content using the `images` and `tables` fields.
:::
