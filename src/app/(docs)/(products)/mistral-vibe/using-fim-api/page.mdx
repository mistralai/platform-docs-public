# Coding

Large language models (LLMs) generate text effectively, and they also perform well at code generation: from code completion to agentic tool use for semi-automated software development.

We provide two major model families for coding:
- **Codestral**: trained for code generation and fill-in-the-middle (FIM) completion.
- **Devstral**: trained for agentic tool use in software development.

We also provide **Codestral Embed** for semantic search and embedding code databases, repositories, and retrieval-powered coding assistants. Learn more in the [code embeddings guide](/studio-api/knowledge-rag/embeddings#code-embeddings).

<SectionTab as="h1" sectionId="before-you-start">Before you start</SectionTab>

## Endpoints and models

We provide two main endpoints:
- FIM: `https://api.mistral.ai/v1/fim/completions`
  - for code completion and code generation; supporting `codestral-latest`.
- Instruction following: `https://api.mistral.ai/v1/chat/completions`
  - for coding and agentic tool use; supporting `codestral-latest`, `devstral-small-latest` and `devstral-medium-latest`.
