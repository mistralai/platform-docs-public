# Platform Overview

Mistral AI has four interfaces, each designed for a different role. This page compares their features so you can pick the right tool for your workflow.

- **Le Chat**: AI workspace for chat, search, document analysis, and agents (no code required)
- **Studio**: developer console for API keys, model testing, fine-tuning, and evaluation
- **Mistral Vibe**: open-source, terminal-native coding agent powered by Devstral
- **Admin**: control plane for organization setup, billing, SSO, and team management

<SectionTab as="h2" sectionId="common-questions">Common questions</SectionTab>

<Faq type="multiple">
  <FaqItem question="What's the difference between Le Chat and Studio?">

Le Chat is for working with AI through conversation (chat, document analysis, agents, and research) without writing code. Studio is for developers building AI applications, providing API key management, a Playground, fine-tuning, and evaluation tooling. Most organizations use both. Business users work in Le Chat, and developers work in Studio.

  </FaqItem>
  <FaqItem question="Can a developer also use Le Chat?">

Yes. Le Chat and Studio share the same account. Developers often use Le Chat for quick testing and ideation before building the same workflow into an API application via Studio.

  </FaqItem>
  <FaqItem question="How does Mistral Vibe fit with Le Chat and Studio?">

Mistral Vibe is a terminal-native coding agent for developers who prefer working in the CLI. It complements Studio for API keys and model management, and Le Chat for browser-based chat. Use Mistral Vibe when you want AI assistance directly in your terminal or IDE.

  </FaqItem>
  <FaqItem question="Who should have the Billing role?">

Only the person responsible for financial operations, typically a finance team member or office manager. The Billing role grants access to subscriptions, invoices, and payment methods without exposing organization settings or user management. Most team members should be Admins or Members.

  </FaqItem>
  <FaqItem question="Do agents built in Le Chat work the same as agents built via the Agents API?">

They use the same underlying model infrastructure, but they're managed separately. Le Chat agents are configured through the UI and shared within your workspace. Agents built via the Agents API in Studio are managed programmatically and are better suited for production deployments where you need version control and automated updates.

  </FaqItem>
</Faq>
