# Interpréteur de code

:::warning
**À noter.** Il s'agit d'une fonctionnalité héritée, toujours disponible dans l’onglet **Chat**. Elle fonctionne comme avant, pas d'urgence. Consultez [Premiers pas avec Work](/vibe/work/get-started) pour accéder au point d’entrée.

Vibe Work intègre un **environnement TypeScript** pour couvrir la plupart des besoins en traitement et visualisation de données (nettoyage, création de graphiques, transformation de fichiers). Privilégiez Work si TypeScript convient ; utilisez Chat avec Interpréteur de code si vous avez besoin de Python ou de bibliothèques exclusives à Python.

L’Interpréteur de code reste disponible pour l’instant. Aucune date de fin annoncée — nous vous tiendrons informé.
:::

L’Interpréteur de code permet de **lancer du Python directement dans Vibe**.

Vous n’êtes pas développeur ? Aucun problème. Décrivez simplement ce que vous voulez et Vibe génère et exécute le code pour vous.

:::info
Les scripts sont exécutés dans un **environnement sécurisé, isolé** et sans accès à Internet. Il ne peut pas accéder à votre système ni à des services externes. Les bibliothèques courantes d’analyse de données (pandas, numpy, matplotlib) sont préinstallées. Les fichiers envoyés ne sont accessibles qu’au cours de la conversation en cours.
:::

<SectionTab as="h2" sectionId="use-cases">Cas d’usage fréquents</SectionTab>

L’Interpréteur de code associe la polyvalence de Vibe à la précision de Python. Parfait pour :

- **Analyse de données** : explorer des fichiers CSV, XLSX ou JSON. Résumer, filtrer, repérer des valeurs aberrantes, nettoyer les données manquantes.
- **Graphiques et visualisations** : générer des graphiques en courbes, en barres, des camemberts ou toute visualisation Matplotlib à partir de vos données.
- **Calculs** : effectuer des formules, des modèles financiers ou des conversions d’unités avec fiabilité.
- **Transformation de fichiers** : conversion de formats, fusion de jeux de données, export de résultats traités en fichiers téléchargeables.

<SectionTab as="h2" sectionId="activation">Activation</SectionTab>

L’Interpréteur de code doit être activé avant que Vibe puisse l’utiliser.

1. Sélectionnez l’onglet **Chat** dans la barre latérale.
2. Cliquez sur l’icône `+` ou saisissez `/` dans la fenêtre de chat.
3. Sélectionnez `Tools` puis activez `Code Interpreter`.

L’outil reste disponible le temps de la conversation et pour les discussions qui suivent.

:::note
L’utilisation de l’Interpréteur de code est **soumise à des limites de débit** selon votre offre. Pour plus d’informations, consultez notre [page tarifs](https://mistral.ai/pricing).
:::

<SectionTab as="h2" sectionId="running-code">Exécution de code</SectionTab>

Décrivez ce que vous souhaitez **en langage naturel**, Vibe écrit et exécute le code Python pour vous.

```text
Chargez ce CSV, affichez les 5 premières lignes, puis tracez les ventes par mois.
```

Le code s’exécute dans la sandbox et renvoie tableaux, graphiques ou fichiers en ligne dans la discussion. Vous pouvez affiner les résultats, ajuster des filtres ou demander une explication du fonctionnement du code.

:::info
L’Interpréteur de code ne peut pas directement récupérer de fichiers depuis une URL. Si vos données sont en ligne, téléchargez-les d’abord puis importez le fichier. Sinon, activez la [Recherche web](/vibe/work/web-search-open-url) et demandez à Vibe de récupérer les données avant d’exécuter le code.
:::

<SectionTab as="h2" sectionId="working-with-files">Travailler avec des fichiers</SectionTab>

Importez vos fichiers à l’aide du bouton `+` dans la barre d’outils du chat. Les formats pris en charge incluent :

- **Documents** : PDF, Word (`.docx`, `.doc`), PowerPoint (`.pptx`, `.ppt`), ODT, EPUB, RTF
- **Tableurs** : Excel (`.xlsx`, `.xls`), CSV, ODS, Numbers
- **Images** : PNG, JPEG, WebP, GIF
- **Textes et formats de balisage** : TXT, Markdown, RST, LaTeX
- **Formats de données** : JSON, JSONL, XML, YAML
- **Fichiers de code** : Python, JavaScript/TypeScript, Java, Go, Rust, C/C++, Ruby, PHP, SQL, et bien d’autres
- **E-mails** : EML, MSG

Exemples de requêtes :

- *« Résume ce tableur et signale les valeurs aberrantes. »*
- *« Nettoie ce CSV, corrige les valeurs manquantes et exporte un nouveau fichier. »*
- *« Extrait tous les e-mails de ce PDF et donne-moi un CSV. »*

:::note
Pour plus de détails et d’exemples avancés, consultez l’article [Fichiers et Canvas](/vibe/work/files-and-canvas).
:::

<SectionTab as="h2" sectionId="tips-for-better-results">Astuces pour de meilleurs résultats</SectionTab>

- **Soyez précis.** Pour les graphiques ou les tableaux, indiquez les noms de colonnes, les filtres et le format de sortie souhaité.
- **Demandez des explications.** Essayez *« Explique les étapes et montre le code »* pour mieux comprendre ce qui se passe.
- **Itérez.** Reprenez la conversation pour affiner le résultat (par exemple, *« Ajoute maintenant une moyenne glissante et nomme les axes. »*)
- **Prototyper avec des petits fichiers.** Si vous atteignez une limite, chargez un échantillon pour tester puis passez à plus grand.
- **Ne partagez aucune donnée sensible.** Ne transmettez jamais vos API keys ou identifiants dans la discussion ou dans du code.
