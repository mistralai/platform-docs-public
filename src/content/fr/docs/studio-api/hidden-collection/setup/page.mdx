# Configuration de collection masquée

Cette page vérifie que les pages enfants d’une catégorie masquée restent accessibles depuis une URL directe.

## Configuration du test

1. Lancez le serveur de documentation local.
2. Ouvrez `/studio-api/hidden-collection/setup` directement.
3. Vérifiez que la page s’affiche et n’apparaît pas dans la barre latérale.

## Résultat attendu

La page doit se charger depuis son URL directe et rester absente des surfaces de navigation.
