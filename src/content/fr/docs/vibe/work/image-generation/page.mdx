# Générer des images

Work peut **générer et modifier des images** directement depuis l’interface de chat. Décrivez ce dont vous avez besoin **en langage courant**, et obtenez des visuels à utiliser dans des présentations, des maquettes, des supports marketing, et plus encore.

:::note
Work utilise les modèles de [Black Forest Labs](https://blackforestlabs.ai) pour la génération d’images.
:::

<SectionTab as="h2" sectionId="activation">Activation</SectionTab>

1. Cliquez sur l’icône `+` ou tapez `/` dans la fenêtre de chat.
2. Sélectionnez `Outils` puis activez `Génération d’images`.

La génération d’images reste active pendant la session en cours. Vous pouvez activer d’autres outils simultanément.

:::info
Des limites de génération quotidiennes s’appliquent selon votre formule. Voir les détails sur [notre page de tarification](https://mistral.ai/pricing).
:::

<SectionTab as="h2" sectionId="creating-images">Création d’images</SectionTab>

Décrivez l’image souhaitée **en langage naturel**. Aucun prompt complexe ni syntaxe particulière n’est nécessaire. Indiquez simplement ce que vous cherchez, Work s’occupe du reste.

Pour de meilleurs résultats, précisez :

- **Sujet** : ce que doit montrer l’image.
- **Style** : illustration, photoréaliste, schéma, flat design, etc.
- **Contraintes** : format / orientation.

Exemple de prompt : *« Créer une illustration plate d’un intérieur de bureau moderne avec de la lumière naturelle et des plantes. »*

<SectionTab as="h2" sectionId="editing-images">Modification d’images</SectionTab>

Vous pouvez affiner les images générées ou modifier des images existantes de deux façons.

<SectionTab as="h3" variant="secondary" sectionId="continue-same-chat">Continuer dans le même chat</SectionTab>

Après avoir généré une image, décrivez les modifications souhaitées **dans un nouveau message**. Work comprend que vous faites référence à l’image précédente et crée une version mise à jour.

Exemple : *« Ajouter un tableau blanc sur le mur du fond. Garder l’éclairage et la palette de couleurs inchangés. »*

<SectionTab as="h3" variant="secondary" sectionId="upload-new-chat">Télécharger une image dans un nouveau chat</SectionTab>

Commencez un nouveau chat, puis téléchargez une image via le bouton `+` en choisissant `Télécharger des fichiers`.

Décrivez les modifications souhaitées, et Work génère une nouvelle version selon vos instructions.

:::tip
C’est utile si vous souhaitez reprendre un résultat précédent ou modifier une image provenant d’une autre source.
:::

<SectionTab as="h2" sectionId="prompt-templates">Modèles de prompts</SectionTab>

Voici des modèles réutilisables pour les cas d’usage les plus courants :

| Type | Modèle |
|------|--------|
| **Scène** | Sujet, style (plat/minimal), composition, format |
| **Photographique** | Sujet, style photoréaliste, type de lumière, objectif, format |
| **Maquette produit** | Sujet, photo studio, arrière-plan, composition, format |
| **Schéma** | Sujet, lignes épurées, labels lisibles, couleurs accessibles, format |
| **Visuel de présentation** | Sujet, composition adaptée aux slides, couleurs de la marque, texte minimal, format |
| **Support marketing** | Sujet, ton adapté à la cible, style, charte graphique, format |

<SectionTab as="h2" sectionId="use-cases">Cas d’usage courants</SectionTab>

- **Visuels pour pitchs et slides** : images principales, séparateurs de sections, fonds stylisés pour présentations investisseurs ou internes.
- **Ressources marketing** : bannières de campagnes, visuels de pages d’accueil, posts réseaux sociaux avec un ton de marque cohérent.
- **Maquettes produit** : mettre un concept en situation sur un fond neutre pour comparer différentes directions avant un passage chez un designer.
- **Communications internes** : images d’en-tête pour annonces, supports d’onboarding, récapitulatifs internes.
- **Exploration de concepts** : générer plusieurs pistes visuelles à partir d’un brief avant de valider une direction.
- **Schémas et illustrations conceptuelles** : diagrammes de processus, organigrammes, abstractions pour des documents et rapports.
- **Images provisoires lors des brouillons** : obtenez un visuel de remplacement immédiat pendant la création d’une ressource finale.

<SectionTab as="h2" sectionId="best-practices">Bonnes pratiques</SectionTab>

- **Précisez votre demande.** *« Un visuel principal photoréaliste pour une landing page fintech, fond sombre, traînées lumineuses abstraites »* produira un bien meilleur résultat que *« faire une bannière »*.
- **Faites plusieurs essais.** Considérez le premier résultat comme un point de départ. Affinez-le avec des prompts complémentaires jusqu’à ce que le résultat vous convienne.
- **Combinez avec des outils pro.** Pour des réglages fins (recadrage précis, corrections colorimétriques, ajout de la charte graphique), associez la génération d’images à un logiciel dédié d’édition.
- **Téléchargez et réutilisez.** Pour enregistrer une image, cliquez sur le bouton de téléchargement.

:::tip
Pour les graphiques, diagrammes et visualisations de données, utilisez plutôt [Code Interpreter](/vibe/chat-legacy/code-interpreter) dans Chat. Il génère des visuels fiables depuis vos données en Python.
:::

<SectionTab as="h2" sectionId="limitations">Limites</SectionTab>

- Les prompts ou images contenant du contenu protégé ou inapproprié peuvent être bloqués par les filtres de sécurité.
- Les détails fins (petits textes, motifs complexes) peuvent parfois être perdus ou modifiés lors de l’édition.
