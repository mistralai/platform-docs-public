## <Pill variant="primary"><b>GET</b></Pill> /v1/workflows/deployments/\{name\}/logs/stream \{#operation-stream_deployment_logs\}

</OperationTitleSection>

<OperationSummarySection slot="summary">

_Stream Deployment Logs_

</OperationSummarySection>

<OperationDescriptionSection slot="description">

Stream logs for a deployment (all of its workers) via SSE.

Resume cursor comes from the `Last-Event-ID` header or `last_event_id` query param (header wins)
and takes precedence over `after`; omit all to tail from the deployment start.

</OperationDescriptionSection>
