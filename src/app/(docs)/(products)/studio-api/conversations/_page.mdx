# Core API

Interact with Mistral models through a unified API for text generation, vision understanding, and structured data extraction. Whether you need simple chat completions or advanced multi-modal workflows, these endpoints provide the foundation for building AI-powered applications.

## Capabilities

- [**Chat Completion**](/studio-api/conversations/chat-completion): Generate text responses from natural language prompts using the `/v1/chat/completions` endpoint. Supports streaming, stop tokens, safe prompting, and prefix-based generation.

- [**Vision**](/studio-api/conversations/vision): Process and understand images alongside text. Compare images, transcribe documents, extract data from charts, and perform OCR on historical documents.

- [**Reasoning**](/studio-api/conversations/reasoning): Use reasoning models that think through problems step by step before answering, improving accuracy on complex tasks like math and coding.

- [**Citations**](/studio-api/conversations/citations): Get source references in model responses, enabling verifiable and traceable AI outputs grounded in your provided context.

- [**Structured Output**](/studio-api/conversations/structured-output): Constrain model outputs to valid JSON schemas or use JSON mode for predictable, machine-readable responses that integrate directly into your application logic.

- [**Function Calling**](/studio-api/conversations/function-calling): Connect models to external tools and APIs by defining JSON schemas for your functions, enabling dynamic data retrieval and action execution.

### Advanced

- [**Predicted Outputs**](/studio-api/conversations/advanced/predicted-outputs): Speed up responses when you can anticipate part of the model's output, reducing latency for edit-style tasks.
