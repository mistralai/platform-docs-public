# Le Chat

Le Chat is our **conversational AI assistant**, available at <AppLink href="https://chat.mistral.ai" app="le-chat">Le Chat</AppLink>, on [iOS](https://apps.apple.com/us/app/le-chat-by-mistral-ai/id6740410176), and on [Android](https://play.google.com/store/apps/details?id=ai.mistral.chat).

Instead of juggling separate tools for writing, research, coding, or image generation, Le Chat lets you do it all in one conversation.

Whether you're drafting a report, analyzing data, or running research for your team, Le Chat is built to fit into your daily workflow. This page covers what's available and how to get started.

<SectionTab as="h1" sectionId="explore">Explore</SectionTab>

Le Chat brings together tools for conversation, content creation, research, and knowledge management.

<ExplorerTabs>
<ExplorerTab value="conversation" label="Conversation" default>

Text, voice, and different reasoning modes. Pick the one that fits.

- **[Chat](/le-chat/conversation/chat)**: have multi-turn conversations with Mistral models. Share or export any conversation when you're done.
- **[Think mode](/le-chat/conversation/think-mode)**: get step-by-step reasoning powered by thinking models for problems that require deeper reflection (aka Chain of Thoughts).
- **[Voice mode](/le-chat/conversation/voice-mode)**: speak instead of typing. Our audio model handles transcription and supports multiple languages.

</ExplorerTab>
<ExplorerTab value="content-creation" label="Content creation">

Generate and iterate on text, images, and code without leaving the conversation.

- **[Canvas](/le-chat/content-creation/canvas)**: write and refine text, code, spreadsheets, and presentations in a side-by-side editor.
- **[Image generation](/le-chat/image-generation)**: create images from text descriptions using [Black Forest Labs](https://bfl.ai/models) models.
- **[Code Interpreter](/le-chat/content-creation/code-interpreter)**: run Python in a secure, isolated environment for data analysis, charts, and calculations.

</ExplorerTab>
<ExplorerTab value="research-and-analysis" label="Research and analysis">

Search the web, analyze documents, or let Le Chat research a topic for you.

- **[Web search](/le-chat/research-analysis/web-search)**: query the web for current information and get answers with source citations.
- **[Deep Research](/le-chat/research-analysis/deep-research)**: kick off automated, multi-step web research that produces structured reports with citations.
- **[Files upload](/le-chat/research-analysis/files-upload)**: share documents, images, and other files directly in conversations for processing and discussion.
- **[Open URL](/le-chat/research-analysis/open-url)**: paste a link and let Le Chat read and analyze the page for you.

</ExplorerTab>
<ExplorerTab value="knowledge-and-integrations" label="Knowledge and integrations">

Bring your own data, set persistent preferences, and organize your work.

- **[Libraries](/le-chat/knowledge-integrations/libraries)**: build personal knowledge bases from your own documents so Le Chat can draw on them in conversations.
- **[Connectors](/le-chat/knowledge-integrations/connectors)**: pull in data from external sources like Gmail, Google Drive, SharePoint, GitHub, and Notion through pre-built integrations.
- **[Memories](/le-chat/knowledge-integrations/memories)**: let Le Chat learn and recall your preferences, decisions, and context across conversations. You control what's stored.
- **[Custom instructions](/le-chat/knowledge-integrations/custom-instructions)**: set persistent rules that shape how Le Chat responds (tone, format, constraints) across all your conversations.
- **[Projects](/le-chat/knowledge-integrations/projects)**: group related conversations and files into Workspaces to keep things organized.
- **[Agents](/le-chat/knowledge-integrations/agents)**: create specialized assistants that combine models with tools, Connectors, and custom instructions for repeatable workflows.

</ExplorerTab>
</ExplorerTabs>

<SectionTab as="h1" sectionId="getting-started">Getting started</SectionTab>

Open <AppLink href="https://chat.mistral.ai" app="le-chat">Le Chat</AppLink> to start a conversation (no account required). [Sign in](https://v2.auth.mistral.ai/login) to your organization or create a personal account to save your conversations and access all features.

:::info
Le Chat is available on several plans, from Free to Enterprise. See our [pricing page](https://mistral.ai/pricing) for details.
:::
