Nous publions en open source les modèles pré-entraînés et les modèles affinés par instruction. Ces modèles ne sont pas ajustés pour la sécurité, car nous souhaitons permettre aux utilisateurs de tester et d'affiner la modération en fonction de leurs cas d'usage. Pour des modèles plus sûrs, consultez notre [tutoriel sur les garde-fous](/studio-api/safety-moderation).

## Comment exécuter ?
Consultez [mistral-inference](https://github.com/mistralai/mistral-inference/), un package Python pour exécuter nos modèles. Vous pouvez installer `mistral-inference` avec
```
pip install mistral-inference
```
