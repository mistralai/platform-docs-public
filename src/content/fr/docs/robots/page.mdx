## Robots d’indexation de Mistral AI

Mistral AI utilise des robots web, également appelés « robots », et des user-agents pour exécuter des tâches liées à ses produits, soit automatiquement, soit à la demande des utilisateurs. Pour permettre aux webmasters de mieux contrôler l’interaction de leur site et de leur contenu avec l’IA, Mistral AI prend en charge des instructions spécifiques dans le fichier robots.txt.

<SectionTab as="h1" sectionId="mistralai-user">MistralAI-User</SectionTab>

`MistralAI-User` correspond aux **actions utilisateur dans Vibe**. Lorsqu’un utilisateur pose une question à Vibe, celui-ci peut **visiter une page web** pour aider à formuler une réponse et **inclure un lien vers la source dans sa réponse**.

`MistralAI-User` détermine les sites pouvant être consultés dans le cadre de telles demandes. Il **n’est pas utilisé pour explorer le web de façon automatique, ni pour collecter du contenu à des fins d’entraînement de modèles génératifs**.

Pour en savoir plus sur notre politique concernant les robots web utilisés pour l’entraînement de l’IA générative, consultez le [Centre Juridique de Mistral AI](https://legal.mistral.ai/).

<SectionTab as="h2" variant="secondary" sectionId="mistralai-user-info">Informations sur le robot</SectionTab>

Chaîne complète de l’user-agent :
- `Mozilla/5.0 AppleWebKit/537.36 (KHTML, like Gecko; compatible; MistralAI-User/1.0; +https://docs.mistral.ai/robots)`

Toutes les adresses IP publiées :
- https://mistral.ai/mistralai-user-ips.json

<SectionTab as="h1" sectionId="mistralai-index">MistralAI-Index</SectionTab>

`MistralAI-Index` effectue une **exploration automatique** du web à **des seuls fins d’indexation**. Ce robot sert à **indexer le contenu** pour le moteur de recherche de Mistral AI, utilisé pour répondre aux requêtes des utilisateurs dans Vibe.

Le contenu exploré par `MistralAI-Index` **n’est jamais utilisé pour l’entraînement de modèles génératifs**.

Pour plus d’informations sur la politique de Mistral AI concernant les robots web utilisés à des fins d’entraînement de l’IA générative, consultez le [Centre Juridique de Mistral AI](https://legal.mistral.ai/).

<SectionTab as="h2" variant="secondary" sectionId="mistralai-index-info">Informations sur le robot</SectionTab>

Chaîne complète de l’user-agent :
- `Mozilla/5.0 AppleWebKit/537.36 (KHTML, like Gecko; compatible; MistralAI-Index/1.0; +https://docs.mistral.ai/robots)`

Toutes les adresses IP publiées :
- https://mistral.ai/mistralai-index-ips.json
