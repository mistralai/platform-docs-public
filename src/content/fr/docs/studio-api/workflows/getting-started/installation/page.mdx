# Installation

Ce guide vous accompagne dans la configuration de Workflows et la vérification de votre installation.

<SectionTab as="h1" sectionId="prerequisites">Prérequis</SectionTab>

Avant d’installer le SDK Workflows, assurez-vous d’avoir :

1. [Python](https://www.python.org/downloads/) 3.12 ou une version ultérieure installé sur votre machine.
2. Le gestionnaire de paquets [uv](https://docs.astral.sh/uv/getting-started/installation/) installé (`uvx` est inclus avec `uv`).

<SectionTab as="h1" sectionId="install-workflows">Installer Workflows</SectionTab>

Installez le package Workflows depuis PyPI avec uv :

```bash
uv add mistralai-workflows
```

Cela créera un environnement virtuel (si aucun n’existe déjà) et installera Workflows ainsi que ses dépendances principales.

<SectionTab as="h2" variant="secondary" sectionId="installing-with-optional-dependencies">Installation avec dépendances optionnelles</SectionTab>

Le plugin Mistral permet l’intégration native des modèles d’IA et services Mistral, dont [les agents durables](/studio-api/workflows/building-workflows/durable_agents), [l’appel d’outils](/studio-api/workflows/building-workflows/durable_agents#built-in-tools) et [la collaboration multi-agent](/studio-api/workflows/building-workflows/durable_agents#multi-agent-handoffs) :

```bash
uv add "mistralai-workflows[mistralai]"
```

Pour [le déport de charges utiles](/studio-api/workflows/building-workflows/payload_offloading) et l’accès direct au stockage cloud depuis vos activités, installez la dépendance supplémentaire adaptée à votre fournisseur :

```bash
# Prise en charge AWS S3
uv add "mistralai-workflows[s3]"

# Prise en charge Azure Blob Storage
uv add "mistralai-workflows[azure]"

# Prise en charge Google Cloud Storage
uv add "mistralai-workflows[gcs]"

# Tous les fournisseurs de stockage
uv add "mistralai-workflows[storage]"
```

<SectionTab as="h1" sectionId="verify-installation">Vérifier l’installation</SectionTab>

Pour vérifier que l’installation a réussi, lancez la commande suivante :

```bash
uv run python -c "import mistralai.workflows; print('Workflows is installed successfully!')"
```
