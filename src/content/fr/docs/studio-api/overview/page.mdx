# Studio

<AppLink href="https://console.mistral.ai" app="studio">Studio</AppLink> est la plateforme développeur de Mistral pour les applications d'IA. Utilisez l'API pour créer des IA conversationnelles, des agents autonomes, de l'intelligence documentaire et des applications RAG. Utilisez la console pour gérer des briques réutilisables, les clés API, tester des modèles et suivre votre utilisation.

<SectionTab as="h1" sectionId="studio-capabilities">Fonctionnalités de Studio</SectionTab>

Studio donne un accès programmatique aux modèles de Mistral pour la génération de texte, les agents, le traitement de données et plus encore.

<SectionTab as="h1" sectionId="getting-started-api">Premiers pas avec l'API</SectionTab>

1. **Créez une clé API** dans la console ou via la page [Clés API](/admin/security-access/api-keys) dans Admin.
2. **Envoyez votre première requête :** suivez le guide de démarrage [envoyer votre première requête API](/getting-started/quickstarts/developer/first-api-request) pour faire un appel chat completion en Python ou TypeScript.

:::tip
Pour approfondir, nous vous recommandons de commencer par le guide [chat completion](/studio-api/conversations/chat-completion), puis d'explorer les autres fonctionnalités selon vos besoins.
:::

<SectionTab as="h1" sectionId="getting-started-console">Premiers pas avec la console</SectionTab>

Ouvrez <AppLink href="https://console.mistral.ai" app="studio">Studio</AppLink> et rendez-vous dans le **Playground** pour tester des prompts, comparer des modèles et expérimenter des paramètres—aucune ligne de code requise. De là, vous pouvez gérer vos clés API, suivre l'utilisation et configurer des agents.

:::info
Certaines fonctionnalités dépendent de votre plan. Consultez la [page des tarifs](https://mistral.ai/pricing) pour plus de détails.
:::
