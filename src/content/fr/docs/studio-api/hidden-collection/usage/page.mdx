# Utilisation d’une collection masquée

Cette page vérifie que les pages enfants masquées peuvent contenir du contenu de documentation standard.

## Envoyer une requête de test

Utilisez une requête de test pour valider le comportement de la page :

```bash
curl https://example.com/hidden-collection/test
```

## Résultat attendu

La page doit afficher le bloc de code et rester exclue des surfaces de découverte.
