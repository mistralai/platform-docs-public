# Panneau Admin

Le <AppLink href="https://admin.mistral.ai" app="admin">Panneau Admin</AppLink> est l’endroit où vous gérez les paramètres, utilisateurs, facturation et sécurité de votre organisation. C’est le centre névralgique pour tout ce qui touche à l’administration sur Vibe et Studio.

<SectionTab as="h1" sectionId="access">Accéder au panneau Admin</SectionTab>

Vous pouvez accéder au panneau Admin de deux façons :

- **URL directe** : rendez-vous sur <AppLink href="https://admin.mistral.ai" app="admin">Panneau Admin</AppLink>.
- **Depuis Studio ou Vibe** : ouvrez le menu principal et sélectionnez **Admin**.

Vous n’avez pas besoin de vous reconnecter. Les sessions sont partagées entre Vibe, Studio et le panneau Admin : la navigation entre les produits est immédiate.

<SectionTab as="h1" sectionId="admin-vs-studio">Panneau Admin vs Studio</SectionTab>

Le panneau Admin et Studio ont des usages différents :

- <AppLink href="https://admin.mistral.ai" app="admin">Panneau Admin</AppLink> : pour la configuration et l’administration. Gérez la facturation, les abonnements, le SSO, les invitations, les journaux d’audit et les paramètres de l’organisation.
- <AppLink href="https://console.mistral.ai" app="studio">Studio</AppLink> : pour concevoir et utiliser des outils. Créez des clés API, gérez les membres de l’espace de travail et exécutez des tâches par lot.

<SectionTab as="h1" sectionId="roles">Accès selon le rôle</SectionTab>

Ce que vous voyez dans le panneau Admin dépend de votre rôle dans l’organisation :

| Rôle | Accès disponible |
|------|-----------------|
| **Admin** | Contrôle total : paramètres de l’organisation, gestion des membres, sécurité, facturation, confidentialité, journaux d’audit |
| **Billing** | Abonnements, factures, moyens de paiement et informations d’utilisation. Impossible d’inviter des utilisateurs ou de modifier les paramètres de l’organisation |
| **Member** | Paramètres du profil et préférences. Accès en lecture seule aux informations de l’organisation |

<SectionTab as="h1" sectionId="what-you-can-manage">Gestion possible</SectionTab>

Depuis le panneau Admin, les administrateurs peuvent :

- **Organisation** : renommer, configurer les paramètres, afficher l’ID de l’organisation
- **Membres** : envoyer une invitation, attribuer des sièges et des rôles, retirer des membres
- **Espaces de travail** : créer des environnements isolés avec clés API et suivi d’utilisation indépendants
- **Connecteurs** : connectez des bots au niveau de l’organisation et contrôlez l’accès aux outils Connecteur
- **Accès** : configurer le SSO et l’authentification par domaine email
- **Facturation** : gérer les moyens de paiement, abonnements et factures
- **Limites d’utilisation** : définir des plafonds de dépenses par espace de travail
- **Journaux d’audit** : surveiller l’activité à l’échelle de l’organisation (Enterprise)
- **Confidentialité** : configurer le traitement des données et les paramètres d’exclusion pour l’entraînement
