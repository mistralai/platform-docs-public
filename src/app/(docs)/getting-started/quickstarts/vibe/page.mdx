# Mistral Vibe

Generate code, edit files, and automate tasks directly from your terminal with <AppLink href="/mistral-vibe/overview" app="vibe">Mistral Vibe</AppLink>.

<CtaButton href="/getting-started/quickstarts/vibe/install-and-first-prompt">Install Mistral Vibe</CtaButton>

- **Terminal-first workflow**: run Mistral Vibe from any project directory
- **Agentic code generation**: describe what you need and Mistral Vibe writes the code
- **Guardrails and permissions**: review every change before it happens
- **Project context awareness**: Mistral Vibe reads your codebase and responds in context

Start with the install quickstart to get Mistral Vibe running, then try scaffolding a full project from a natural-language description.
