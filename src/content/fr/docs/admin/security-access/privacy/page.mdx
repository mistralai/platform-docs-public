# Confidentialité

Vous disposez de contrôles pour gérer l’utilisation de vos données dans Vibe et via l’API. Cette page couvre nos politiques d’utilisation des données, le refus de participation à l’entraînement, et vos droits au titre du RGPD.

<SectionTab as="h1" sectionId="data-usage">Utilisation des données</SectionTab>

- **Vibe (Free plan)** : vos conversations peuvent être utilisées pour améliorer nos modèles. Vous pouvez retirer votre consentement à tout moment depuis le panneau Admin.
- **Vibe (Pro, Team, Enterprise)** : vos conversations ne sont pas utilisées pour l’entraînement des modèles par défaut.
- **API** : les données envoyées via l’API ne sont pas utilisées pour l’entraînement.
- **Connectors** : les données accédées via les Connectors sont récupérées à la demande et ne sont pas stockées de façon permanente. Elles ne servent pas à l’entraînement.
- **Libraries** : les documents importés sont stockés de manière sécurisée et ne sont pas utilisés pour l’entraînement.

<SectionTab as="h1" sectionId="opt-out">Refus de participation à l’entraînement</SectionTab>

Pour refuser l’utilisation de vos données à des fins d’amélioration des modèles :

1. Ouvrez <AppLink href="https://admin.mistral.ai" app="admin">Confidentialité</AppLink>.
2. Désactivez le partage des données pour l’entraînement des modèles.

Le changement est immédiat pour votre organisation.

<SectionTab as="h1" sectionId="gdpr">Droits RGPD</SectionTab>

Selon le RGPD, vous disposez des droits suivants :

| Droit | Description |
|-------|-------------|
| **Accès** | Obtenir une copie de vos données personnelles |
| **Rectification** | Corriger les informations personnelles inexactes |
| **Effacement** | Supprimer votre compte et les données associées |
| **Portabilité** | Obtenir vos données dans un format transférable |
| **Opposition** | Refuser certains traitements spécifiques |
| **Limitation** | Demander la restriction de certains traitements |

Pour exercer vos droits, utilisez notre [formulaire de contact dédié](https://help.mistral.ai). Veillez à utiliser l’adresse email associée à votre compte et à préciser les données ou activités concernées.

Pour modifier les données liées à votre compte, utilisez directement le panneau Admin. Pour exporter vos données, téléchargez-les depuis Vibe ou Studio.

:::warning
La suppression du compte est définitive et irréversible. Tous les prompts, réponses et données personnelles seront supprimés, sans possibilité de récupération.
:::

<SectionTab as="h1" sectionId="legal">Documents juridiques</SectionTab>

Pour les conditions d’utilisation, politiques de confidentialité et accords relatifs au traitement des données, consultez le Legal and Trust Center sur [mistral.ai](https://mistral.ai).
