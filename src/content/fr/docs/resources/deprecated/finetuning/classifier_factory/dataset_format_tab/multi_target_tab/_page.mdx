Vous pouvez également avoir plusieurs cibles et pas seulement une seule. Cela est utile si vous souhaitez classifier différents aspects du même contenu de manière indépendante. Exemple :

<Tabs groupId="classification-type">
  <TabItem value="v1/classifications" label="v1/classifications" default>

```json
{
  "text": "I love this product!",
  "labels": {
    "sentiment": "positive", // positive/neutral/negative
    "is-english": "yes" // yes/no, boolean
  }
}
```

  </TabItem>
  <TabItem value="v1/chat/classifications" label="v1/chat/classifications">

```json
{
  "messages": [{ "role": "user", "content": "I love this product!" }],
  "labels": {
    "sentiment": "positive", // positive/neutral/negative
    "is-english": "yes" // yes/no, boolean
  }
}
```

  </TabItem>
</Tabs>

- Chaque cible est indépendante des autres, ce qui signifie que les scores pour chaque label seront également indépendants.
