# Document processing

Mistral Document AI offers enterprise-level document processing, combining OCR technology with advanced structured data extraction.

<Image
  url={['/img/document_ai_overview.png', '/img/document_ai_overview_dark.png']}

/>

Use multilingual support, annotations, and adaptable workflows for all document types to extract, comprehend, and analyze information.

<SectionTab as="h1" sectionId="document-ai-services">Document AI services</SectionTab>

Use `client.ocr.process` in our SDK clients or the `https://api.mistral.ai/v1/ocr` endpoint to access the following services:

- [OCR Processor](/studio-api/document-processing/basic_ocr): Discover our OCR model and its capabilities.
- [Annotations](/studio-api/document-processing/annotations): Annotate and extract data from your documents using built-in structured outputs.
- [Document QnA](/studio-api/document-processing/document_qna): Combine our models with OCR technology for document question answering.
