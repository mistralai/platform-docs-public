# Dépannage de collection masquée

Cette page vérifie que les pages imbriquées masquées conservent leurs métadonnées et leurs titres.

## La page renvoie une erreur 404

Si cette page renvoie une erreur 404, vérifiez que les réécritures de la locale par défaut incluent toujours les routes masquées.

## La page apparaît dans la navigation

Si cette page apparaît dans la navigation, vérifiez le filtrage des catégories masquées dans `getSidebar`.
