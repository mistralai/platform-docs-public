# Commandes et raccourcis clavier

Utilisez les **commandes slash** et les **raccourcis clavier** pour contrôler une session Vibe sans quitter le terminal.

<SectionTab as="h2" sectionId="slash-commands">Commandes slash</SectionTab>

Tapez `/` dans l’invite de ligne de commande pour ouvrir le sélecteur de commandes slash. La saisie semi-automatique propose des commandes correspondantes au fur et à mesure de votre saisie. Le tableau ci-dessous répertorie les commandes intégrées.

:::tip
[Skills](/vibe/code/cli/skills) can also expose themselves as slash commands. If you see a `/` command that isn't in this list, it likely comes from a local skill (check `~/.vibe/skills/` or your project's `.vibe/skills/`).
:::

| Commande | Alias | Description |
|---|---|---|
| `/help` |  | Affiche l’aide pour la session CLI en cours. |
| `/config` |  | Modifier les paramètres de configuration. |
| `/model` |  | Sélectionner le modèle actif. |
| `/thinking` |  | Sélectionner le niveau de réflexion. |
| `/reload` |  | Recharger la configuration, les instructions de l’agent et les Skills à partir du disque. |
| `/clear` |  | Effacer l’historique de la conversation. |
| `/copy` |  | Copier le dernier message de l’agent dans le presse-papiers. |
| `/log` |  | Afficher le chemin du fichier journal de l’interaction en cours. |
| `/debug` |  | Activer ou désactiver la console de débogage. |
| `/compact` |  | Condenser l’historique de la conversation en le résumant. Passer des instructions facultatives pour guider le résumé. |
| `/exit` |  | Quitter l’application. |
| `/status` |  | Afficher les statistiques de l’agent. |
| `/teleport` |  | Transférer la session vers Vibe Code Web. |
| `/proxy-setup` |  | Configurer les paramètres de proxy et de certificat SSL. |
| `/resume` | `/continue` | Parcourir et reprendre les sessions précédentes. |
| `/rename` |  | Renommer la session en cours. |
| `/mcp` | `/connectors` | Afficher les serveurs MCP et les Connecteurs disponibles. Passer un nom pour lister ses outils. |
| `/voice` |  | Configurer les paramètres vocaux. |
| `/leanstall` |  | Installer [`leanstral`](https://github.com/mistralai/leanstral), l’agent assistant de preuve Lean 4. |
| `/unleanstall` |  | Désinstaller l’agent Lean 4. |
| `/rewind` |  | Revenir à un message précédent. |
| `/loop` |  | Planifier un prompt récurant. Utilisez `/loop interval prompt`, `/loop list` ou `/loop cancel id` ou `all`. |
| `/data-retention` |  | Afficher les informations de conservation des données. |

<SectionTab as="h2" sectionId="keyboard-shortcuts">Raccourcis clavier</SectionTab>

:::warning
Some shortcuts may not work in every combination of terminal emulator, SSH session, or terminal multiplexer (such as tmux). For the best experience, we recommend [Ghostty](https://ghostty.org/), [Kitty](https://sw.kovidgoyal.net/kitty/), [WezTerm](https://wezterm.org/), or [iTerm2](https://iterm2.com/).
:::

| Raccourci | Action |
|---|---|
| `Ctrl+C` | Si du texte est sélectionné, le copier. Sinon, effacer la saisie. Avec une saisie vide, quitter ou interrompre l’opération en cours. |
| `Ctrl+D` | Supprimer le caractère à droite ou quitter si la saisie est vide. |
| `Ctrl+Z` | Suspendre le CLI. |
| `Escape` | Interrompre l’opération en cours. |
| `Ctrl+O` | Activer ou désactiver l’affichage de la sortie des outils. |
| `Ctrl+Y` | Copier la sélection en cours. |
| `Ctrl+Shift+C` | Copier la sélection en cours. |
| `Shift+Tab` | Passer en revue les modes ou agents disponibles. |
| `Shift+Up` | Faire défiler la conversation vers le haut. |
| `Shift+Down` | Faire défiler la conversation vers le bas. |
| `Ctrl+G` | Modifier le plan actuel dans un éditeur externe. |
| `Ctrl+\` | Activer ou désactiver la console de débogage. |
| `Alt+Up` (`Option+Up` sur macOS) | Revenir au message précédent. |
| `Ctrl+P` | Revenir au message précédent. |
| `Alt+Down` (`Option+Down` sur macOS) | Aller au message suivant après un retour en arrière. |
| `Ctrl+N` | Aller au message suivant après un retour en arrière. |
