# Studio

Build, test, and deploy AI applications with the Mistral API through <AppLink href="https://console.mistral.ai" app="studio">Studio</AppLink>.

<CtaButton href="https://console.mistral.ai">Open Studio</CtaButton>

- **Free Experiment plan**: start making API calls with no credit card required
- **API playground**: test models and adjust parameters in the browser
- **Workspace management**: organize API keys, usage, and team access
- **API key generation**: create scoped keys for your applications and scripts

Start with the activation quickstart to set up your account, or go straight to the playground if you already have an API key.
