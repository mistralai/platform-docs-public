# Vibe

:::info
**Le Chat is now Vibe.** The <AppLink href="https://chat.mistral.ai">chat.mistral.ai</AppLink> URL remains the entry point and most features now live inside **Work**, with a few legacy ones still available in **Chat** (see below). Your plan, history, and settings are preserved in **Chat**.
:::

Vibe is Mistral's **unified agent** for productivity and coding tasks, built for professional use and available across web, mobile, your code editor, and your terminal.

It runs in three modes, each shaped for a different kind of task:

- **Work**: multi-stage professional tasks across your apps and tools (research, drafts, summaries, scheduled runs).
- **Code**: development in your terminal, in VS Code, or as remote cloud sessions.
- **Chat**: quick turn-based conversations and legacy Le Chat features (Agents, Think mode, Deep Research, Code Interpreter, Memories).

:::tip
Need help choosing the right mode for your task? See [Choose Chat, Work, or Code](/vibe/choose-chat-work-code) for a side-by-side comparison.
:::

<SectionTab as="h2" sectionId="how-vibe-works">How Vibe works</SectionTab>

Across every mode, you interact with Vibe **the same way**: same Mistral models, same patterns, same control. A typical Vibe workflow looks like this:

1. **You describe the outcome you want** in natural language.
2. **Vibe gathers context** from your prompt, attached files, connected tools, or the web.
3. **Vibe plans and acts**: it breaks the task into steps, calls the right tools, and shows its progress in real time.
4. **You review and reuse**: outputs are inspectable, editable, and reusable across sessions.

**At every step, you stay in control.** Vibe asks for approval before sensitive actions, shows every tool call, and lets you steer or stop at any point.
