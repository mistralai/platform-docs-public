# Index de recherche

Les backends de stockage conservent les morceaux traités et permettent une recherche efficace dans votre collection de documents.

Le Search Toolkit prend en charge les **vector stores** pour la recherche sémantique. Vous pouvez combiner vector stores et keyword stores pour une récupération hybride, souvent optimale.

<SectionTab as="h1" sectionId="vector-stores">Vector stores</SectionTab>

Les vector stores permettent la **recherche sémantique** en stockant des embeddings de morceaux et en recherchant les vecteurs similaires via des métriques de distance.

### Vespa

Vespa est un moteur de recherche et une base de données vectorielle évolutifs.
- Prend en charge nativement la recherche vectorielle et par mots-clés
- Gère la récupération hybride avec des profils de requête configurables
- Pense pour des charges de production évolutives
- Nécessite un embedder explicite

**Installation** :
```bash
pip install "mistralai-search-toolkit[vespa]"
```

**Exemple** :

```python
from mistralai.search.toolkit.plugins.vespa import VespaClient, VespaConfig

config = VespaConfig(
    collection_name="my_collection",
    query_endpoint="http://localhost:8080",
    feed_endpoint="http://localhost:8080",
)

vector_store = VespaClient(config)
```

<SectionTab as="h1" sectionId="custom-storage-backends">Backends de stockage personnalisés</SectionTab>

Implémentez des backends de stockage personnalisés en suivant le protocole adapté :

### Vector store personnalisé

```python
from mistralai.search.toolkit.indices import VectorStoreClient
from mistralai.search.toolkit.document import Document
from mistralai.search.toolkit.indices import VectorSearchQuery, SearchResult

class MyVectorStore(VectorStoreClient):
    async def create_index(self) -> bool:
        pass

    async def upsert(self, document: Document) -> bool:
        pass

    async def search(self, query: VectorSearchQuery) -> list[SearchResult]:
        pass

    async def delete_document(self, doc_id: str) -> None:
        pass
```

```python
from mistralai.search.toolkit.indices import KeywordStoreClient

        pass

        pass
```
