# Vision

Vision lets our models **analyze images alongside text**. Send a photo, a chart, or a scanned document along with a prompt, and the model responds based on both.

All vision-capable models are available through the [Chat Completions API](./completion/).

:::info
For document parsing, OCR, and structured data extraction, use [Document AI](./document_ai) instead.
:::

<SectionTab as="h1" sectionId="faq">FAQ</SectionTab>

<Faq>
  <FaqItem question="What is the price per image?">
    Images are priced as input tokens. Each image is tokenized based on its resolution.
  </FaqItem>
  <FaqItem question="How many tokens correspond to an image and/or what is the maximum resolution?">

    Tokenization varies by model and resolution:

    | Model             | Max resolution | Approximate formula                   | Approximate max tokens |
    | ----------------- | -------------- | ------------------------------------- | ---------------------- |
    | Mistral Large 3   | 1540x1540      | `≈ (ResolutionX * ResolutionY) / 784` | ≈ 3025                 |
    | Mistral Small 3.2 | 1540x1540      | `≈ (ResolutionX * ResolutionY) / 784` | ≈ 3025                 |
    | Mistral Medium 3  | 1540x1540      | `≈ (ResolutionX * ResolutionY) / 784` | ≈ 3025                 |
    | Ministral 3       | 1540x1540      | `≈ (ResolutionX * ResolutionY) / 784` | ≈ 3025                 |
    | Pixtral Large     | 1024x1024      | `≈ (ResolutionX * ResolutionY) / 256` | ≈ 4096                 |
    | Pixtral 12B       | 1024x1024      | `≈ (ResolutionX * ResolutionY) / 256` | ≈ 4096                 |

    Images larger than the model's maximum resolution are downscaled automatically. The API returns an error if the resolution exceeds **10000x10000**.

  </FaqItem>
  <FaqItem question="Can I fine-tune the image capabilities?">
    Yes, you can fine-tune pixtral-12b.
  </FaqItem>
  <FaqItem question="Can I use them to generate images?">
    No. Vision models analyze and understand images. They don't generate them.
  </FaqItem>
  <FaqItem question="What types of image files are supported?">

    We support the following formats:

    - PNG (.png)
    - JPEG (.jpeg and .jpg)
    - WEBP (.webp)
    - Non-animated GIF with a single frame (.gif)

  </FaqItem>
  <FaqItem question="Is there a limit to the size of the image?">
    The file size limit is 10 MB.
  </FaqItem>
  <FaqItem question="What's the maximum number of images per request?">
    Eight images per API request.
  </FaqItem>
  <FaqItem question="What is the rate limit?">
    Check the [rate limits page](https://console.mistral.ai/limits/) for current values.
  </FaqItem>
</Faq>
