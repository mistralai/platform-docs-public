# Commandes et raccourcis

Utilisez les **slash commands** et les **raccourcis clavier** pour piloter une session Vibe sans quitter le terminal.

<SectionTab as="h2" sectionId="slash-commands">Commandes slash</SectionTab>

Saisissez « / » à l’prompt du CLI pour ouvrir le sélecteur de commandes slash. L’autocomplétion suggère les commandes correspondantes au fur et à mesure de la saisie. Le tableau ci-dessous liste les commandes intégrées.

:::tip
Les [Compétences](/vibe/code/cli/skills) peuvent également apparaître sous forme de commandes slash. Si vous voyez une commande « / » absente de cette liste, elle provient probablement d’une compétence locale (vérifiez `~/.vibe/skills/` ou `.vibe/skills/` dans votre projet).
:::

| Commande | Alias | Description |
|---|---|---|
| `/help` |  | Affiche l’aide pour la session CLI en cours. |
| `/config` |  | Modifier la configuration. |
| `/model` |  | Sélectionner le modèle actif. |
| `/thinking` |  | Choisir le niveau de réflexion. |
| `/reload` |  | Recharger la configuration, les instructions agent et les skills depuis le disque. |
| `/clear` |  | Effacer l’historique de la conversation. |
| `/copy` |  | Copier le dernier message de l’agent dans le presse-papiers. |
| `/log` |  | Afficher le chemin du fichier log de l’interaction actuelle. |
| `/debug` |  | Activer/désactiver la console de debug. |
| `/compact` |  | Résumer l’historique de la conversation. Vous pouvez préciser des instructions pour orienter le résumé. |
| `/exit` |  | Quitter l’application. |
| `/status` |  | Afficher les statistiques de l’agent. |
| `/teleport` |  | Ouvrir la session sur Vibe Code Web. |
| `/proxy-setup` |  | Configurer les paramètres proxy et certificat SSL. |
| `/resume` | `/continue` | Parcourir et reprendre les sessions précédentes. |
| `/rename` |  | Renommer la session en cours. |
| `/mcp` | `/connectors` | Afficher les serveurs et connecteurs MCP disponibles. Ajoutez un nom pour obtenir sa liste d’outils. |
| `/voice` |  | Configurer les paramètres vocaux. |
| `/leanstall` |  | Installer [`leanstral`](https://github.com/mistralai/leanstral), l’agent assistant de preuve Lean 4. |
| `/unleanstall` |  | Désinstaller l’agent Lean 4. |
| `/rewind` |  | Revenir à un message antérieur. |
| `/loop` |  | Programmer un prompt récurrent. Utilisez `/loop interval prompt`, `/loop list` ou `/loop cancel id` ou `all`. |
| `/data-retention` |  | Afficher les informations sur la conservation des données. |

<SectionTab as="h2" sectionId="keyboard-shortcuts">Raccourcis clavier</SectionTab>

:::warning
Certains raccourcis peuvent ne pas fonctionner selon le terminal, la session SSH ou le multiplexeur (comme tmux). Pour une expérience optimale, nous recommandons [Ghostty](https://ghostty.org/), [Kitty](https://sw.kovidgoyal.net/kitty/), [WezTerm](https://wezterm.org/), ou [iTerm2](https://iterm2.com/).
:::

| Raccourci | Action |
|---|---|
| `Ctrl+C` | Si du texte est sélectionné, le copier. Sinon, effacer la saisie. Si la saisie est vide, quitter ou interrompre l’opération en cours. |
| `Ctrl+D` | Supprimer le caractère à droite, ou quitter si l’entrée est vide. |
| `Ctrl+Z` | Suspendre le CLI. |
| `Escape` | Interrompre l’opération en cours. |
| `Ctrl+O` | Activer/désactiver la sortie outil. |
| `Ctrl+Y` | Copier la sélection actuelle. |
| `Ctrl+Shift+C` | Copier la sélection actuelle. |
| `Shift+Tab` | Parcourir les modes ou agents disponibles. |
| `Shift+Up` | Faire défiler le chat vers le haut. |
| `Shift+Down` | Faire défiler le chat vers le bas. |
| `Ctrl+G` | Modifier le plan actuel dans un éditeur externe. |
| `Ctrl+\\` | Activer/désactiver la console de debug. |
| `Alt+Up` (`Option+Up` sur macOS) | Revenir au message précédent. |
| `Ctrl+P` | Revenir au message précédent. |
| `Alt+Down` (`Option+Down` sur macOS) | Aller au message précédent auquel vous êtes remonté. |
| `Ctrl+N` | Aller au message précédent auquel vous êtes remonté. |
