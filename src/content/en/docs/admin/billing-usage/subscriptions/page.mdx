# Subscriptions

Use Subscriptions to review the plans and seats active on your Organization. Admin separates product subscriptions for Vibe and Mistral Code from the API plan used for Studio and API usage.

For the latest plan details and pricing, use the public pricing pages:

- [Vibe pricing](https://mistral.ai/pricing/)
- [API and model pricing](https://mistral.ai/pricing/#api)

<SectionTab as="h1" sectionId="open-subscriptions">Open Subscriptions</SectionTab>

Open <AppLink href="https://admin.mistral.ai/subscriptions" path={["Admin Panel", "Subscriptions", "Plans"]} />.

In Subscriptions, you can review:

- Vibe subscription and seats;
- API Plan, including Free mode and Scale;
- Mistral Code seats;
- pay-as-you-go settings for Vibe CLI.

<SectionTab as="h1" sectionId="vibe-plans">Vibe plans</SectionTab>

Vibe plans control access to Vibe features and seats. In Admin, you can see how many Team seats are used and available.

For plan features, pricing, and current limits, see [Vibe pricing](https://mistral.ai/pricing/).

:::tip
Students and teachers can check whether they are eligible for a discount in the [student and teacher discount article](https://help.mistral.ai/en/articles/698530-i-m-a-student-or-a-teacher-am-i-eligible-for-a-discount).
:::

<SectionTab as="h1" sectionId="api-plan">API Plan</SectionTab>

The API Plan controls Studio and API usage. It is separate from the Vibe subscription.

You can review your API Plan from <AppLink href="https://admin.mistral.ai/subscriptions" path={["Admin Panel", "Subscriptions", "Subscriptions"]} />, in the **API Plan** tab.

<SectionTab as="h2" variant="secondary" sectionId="free-mode">Free mode</SectionTab>

Free mode lets you create API keys and use the free tier within the limits shown on the [Usage and limits](/admin/billing-usage/usage-limits) page. This free usage is included with your Vibe subscription if you have one, and is available by default if you do not.

<SectionTab as="h2" variant="secondary" sectionId="scale-plan">Scale plan</SectionTab>

Scale is the pay-as-you-go API plan. It gives access to paid API usage and higher limits, including frontier models, agents, fine-tuning, and preview or beta features.

For exact model pricing, included limits, and feature availability, see [API and model pricing](https://mistral.ai/pricing/#api).

To upgrade, open the [Scale upgrade page](https://console.mistral.ai/upgrade/subscribe?via=subscriptions).

<SectionTab as="h1" sectionId="mistral-code">Mistral Code and Vibe Code</SectionTab>

Some Organizations still show Mistral Code seats. Mistral Code seats are legacy Enterprise seats for the coding assistant. The current product direction is Vibe Code, which covers the CLI, editor extensions, and coding workflows.

If you need to change the number of Mistral Code Enterprise seats, contact Mistral.
