# Quickstarts

Each quickstart is self-contained: it starts with prerequisites, walks you through every step, and ends with working output you can verify. Quickstarts are grouped by role so you can go directly to the tasks that match your workflow. Most take between 5 and 20 minutes to complete.

- **Le Chat**: draft reports, analyze data, and create agents without code.
- **Studio**: activate the API, generate keys, and test models in the playground.
- **Mistral Vibe**: install the CLI, generate code, and scaffold projects from your terminal.
- **Developer**: set up the SDK, send API requests, and build agents.
- **Admin**: create your organization, configure SSO, and manage workspaces.
