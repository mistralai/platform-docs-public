# Activez Studio et générez une clé API

Créez votre compte <AppLink href="https://console.mistral.ai" app="studio">Studio</AppLink> et créez une clé API à utiliser pour appeler les modèles Mistral.

- **Mode gratuit** : l’accès à l’API est activé par défaut, sans carte bancaire requise. Limites d’usage et de débit applicables.
- **Gestion sécurisée des clés** : définissez des dates d’expiration et effectuez une rotation régulière des clés

À la fin de cette étape, vous aurez une clé API fonctionnelle prête à être utilisée pour votre première requête.

**Temps estimé** : ~5 minutes

<SectionTab as="h2" sectionId="prerequisites">Prérequis</SectionTab>

- Un <AppLink href="https://console.mistral.ai">compte Mistral</AppLink>.

<SectionTab as="h2" sectionId="step-1">Étape 1 : Générer une clé API</SectionTab>

1. Ouvrez la <AppLink href="https://console.mistral.ai" app="studio">console Studio</AppLink>.
2. Accédez à `Clés API` dans la barre latérale de gauche.
3. Cliquez sur **Créez une nouvelle clé**.

![Cliquez sur Créez une nouvelle clé](/assets/quickstarts/studio/new-key-button.png)

1. Ajoutez un `Nom` pour identifier la clé (par exemple, `Première clé de test`).
2. Définissez une date d’`Expiration`. Une rotation régulière améliore la sécurité.
3. Choisissez un `Périmètre d’accès des connecteurs` :
   - `Connecteurs partagés uniquement` : accès uniquement aux Connecteurs partagés avec l’espace de travail, et non à vos connecteurs privés.
   - `Connecteurs privés et partagés` : accès à la fois à vos connecteurs privés et aux Connecteurs partagés avec l’espace de travail.

![Configurez la clé API avec un nom facultatif et une date d’expiration](/assets/quickstarts/studio/new-key-modal.png)

1. Cliquez sur `Créez une nouvelle clé`.
2. Copiez immédiatement la clé et stockez-la dans un endroit sécurisé (gestionnaire de mots de passe ou coffre-fort de secrets).

:::warning
La clé complète n’apparaît qu’une seule fois. Vous ne pourrez plus la retrouver après fermeture de la fenêtre de confirmation.
:::

<SectionTab as="h2" sectionId="verify">Vérifiez</SectionTab>

Vous avez terminé la configuration :

1. Vous êtes en mode gratuit avec des limites d’usage et de débit appliquées.
2. Vous avez une clé API stockée en toute sécurité.

Testez la clé avec une requête `curl` rapide :

```bash
curl https://api.mistral.ai/v1/chat/completions \
  -H "Authorization: Bearer $MISTRAL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "mistral-small-latest",
    "messages": [{"role": "user", "content": "Hello, Mistral!"}]
  }'
```

Une réponse réussie confirme le fonctionnement de votre clé.
