# Agents

Transformez les modèles Mistral en véritables résolveurs de problèmes actifs. L'API Agents combine des modèles de langage puissants avec des outils intégrés pour l'exécution de code, la recherche web, la génération d'images et les intégrations externes, permettant un raisonnement multi-étapes et l'accomplissement autonome de tâches.

## Dans cette section

- [**Introduction**](/studio-api/agents/introduction) : Démarrez avec les agents, comprenez les concepts fondamentaux et découvrez en quoi les agents diffèrent des chat completions standard.

- [**API Agents**](/studio-api/agents/agents-api) : Créez, configurez et gérez des agents par programmation. Gérez les conversations, le streaming, les ramifications et les opérations du cycle de vie des agents.

- [**Outils d'agents**](/studio-api/agents/agent-tools) : Équipez les agents de capacités : outils intégrés (recherche web, interpréteur de code, génération d'images, bibliothèques de documents), appel de fonctions personnalisées et l'API d'appel de fonctions pour une utilisation structurée des outils.

- [**Handoffs**](/studio-api/agents/handoffs) : Orchestrez plusieurs agents spécialisés pour collaborer sur des tâches complexes, en transférant le contrôle entre eux selon le problème à traiter.
