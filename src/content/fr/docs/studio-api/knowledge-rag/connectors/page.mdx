# Connecteurs

:::info
Les connecteurs sont une fonctionnalité en **bêta**. L'interface de l'API peut évoluer.
:::

Les connecteurs sont des [serveurs MCP](https://modelcontextprotocol.io/) enregistrés que vous pouvez utiliser comme **outils dans les conversations et les Agents**, depuis n'importe quel SDK ou directement via l'API, **sans gérer le transport MCP localement**.

Une fois enregistré, un connecteur **expose ses outils au modèle** à la demande : le modèle les découvre automatiquement et appelle le bon **en fonction de la requête de l'utilisateur**.

:::tip[Qu'est-ce que MCP ?]
Le [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) est un standard ouvert permettant de connecter les modèles d'IA à des outils et sources de données externes via une interface unifiée. Tout **serveur compatible MCP** peut être enregistré comme connecteur.
:::
