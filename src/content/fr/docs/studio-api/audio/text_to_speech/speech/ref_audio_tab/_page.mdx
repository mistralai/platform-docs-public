<SectionTab as="h2" sectionId="ref-audio-guidelines">Recommandations pour `ref_audio`</SectionTab>

Lors de l'utilisation de `ref_audio` pour le clonage zero-shot :

- **Durée** : 3 à 25 secondes
- **Un seul locuteur**
- **Enregistrement propre** sans bruit de fond
- **Prosodie neutre** — évitez les pauses excessives ou les disfluences
- **Intonation expressive** (les voix monotones produisent un résultat monotone)
