Un autre cas d'usage potentiel est la détection de paraphrases. Dans cet exemple simple, nous avons une liste de trois phrases, et nous souhaitons déterminer si deux d'entre elles sont des paraphrases l'une de l'autre. Si la distance entre deux embeddings de phrases est faible, cela suggère que les deux phrases sont sémantiquement similaires et pourraient être des paraphrases potentielles.

Le résultat suggère que les deux premières phrases sont sémantiquement similaires et pourraient être des paraphrases potentielles, tandis que la troisième phrase est plus différente. Il s'agit d'un exemple très simple. Mais cette approche peut être étendue à des situations plus complexes dans des applications réelles, comme la détection de paraphrases dans les publications sur les réseaux sociaux, les articles de presse ou les avis clients.

```python
import itertools

sentences = [
    "Have a safe happy Memorial Day weekend everyone",
    "To all our friends at Whatsit Productions Films enjoy a safe happy Memorial Day weekend",
    "Where can I find the best cheese?",
]

sentence_embeddings = [get_text_embedding([t]) for t in sentences]

sentence_embeddings_pairs = list(itertools.combinations(sentence_embeddings, 2))
sentence_pairs = list(itertools.combinations(sentences, 2))
for s, e in zip(sentence_pairs, sentence_embeddings_pairs):
    print(s, euclidean_distances([e[0]], [e[1]]))
```
