# Connaissances & RAG

Connectez vos données aux modèles Mistral pour la génération augmentée par récupération. Des embeddings vectoriels aux connecteurs de données externes, ces outils vous permettent d'ancrer les réponses du modèle dans vos propres bases de connaissances, documents et sources de données.
