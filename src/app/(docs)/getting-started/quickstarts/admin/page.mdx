# Admin

Administrative tasks for running Mistral in your organization, all managed through Admin.

<CtaButton href="https://admin.mistral.ai">Open Admin</CtaButton>

- **Create your organization**: set up billing, invite team members
- **Configure SSO**: verify your domain and connect your identity provider
- **Manage workspaces**: isolate API keys, usage, and spending by team

Start with organization creation if you're new, or jump to SSO or workspace management if your account is already active.
