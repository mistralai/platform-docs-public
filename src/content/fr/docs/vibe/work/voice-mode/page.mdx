# Utiliser le mode Voix

Le mode Voix vous permet de lancer une tâche dans Work en utilisant votre voix au lieu de taper. Votre message vocal est transcrit en texte, que vous pouvez relire, modifier et envoyer comme un prompt. Cette fonctionnalité est disponible sur l’application web et les applications mobiles ([iOS](https://apps.apple.com/us/app/le-chat-by-mistral-ai/id6740410176) et [Android](https://play.google.com/store/apps/details?id=ai.mistral.chat)) dans tous les plans.

:::info
Le mode Voix est basé sur notre [série de modèles Voxtral](https://mistral.ai/news/voxtral-transcribe-2), un modèle de reconnaissance vocale que nous avons développé pour une transcription rapide et fiable dans plusieurs langues.
:::

<SectionTab as="h2" sectionId="how-to-use">Comment l'utiliser</SectionTab>

1. Cliquez sur l’icône micro en bas à droite de la fenêtre de discussion.
2. Autorisez l’accès au micro lorsque votre navigateur ou appareil le demande.
3. Parlez normalement.

La suite dépend du paramètre `Autosend` :

- **Autosend ACTIVÉ** : votre voix est transcrite puis envoyée automatiquement à Work. Aucune action supplémentaire n’est requise.
- **Autosend DÉSACTIVÉ** : le texte transcrit apparaît dans la zone de saisie. Vous pouvez le relire, corriger d’éventuelles erreurs, joindre du contexte (Connecteurs, Bibliothèques, Compétences, fichiers), puis cliquer sur `envoyer` quand vous êtes prêt.

Pour supprimer un enregistrement, cliquez sur l’icône annuler avant l’envoi.

:::tip
Si vous avez précédemment refusé l’accès au micro, activez-le dans les paramètres du site du navigateur ou dans les réglages de l’application sur votre appareil.
:::

<SectionTab as="h2" sectionId="supported-languages">Langues prises en charge</SectionTab>

Le mode Voix prend actuellement en charge les langues suivantes :

- Anglais
- Français
- Espagnol
- Allemand
- Russe
- Chinois
- Japonais
- Italien
- Portugais
- Néerlandais
- Arabe
- Hindi
- Coréen

<SectionTab as="h2" sectionId="privacy">Confidentialité</SectionTab>

Le traitement de vos données vocales dépend de votre plan :

- **Plans Free, Pro et Student** : les messages vocaux sont soumis à notre politique de collecte de données et peuvent être utilisés pour améliorer nos modèles. Vous pouvez [vous opposer à l’utilisation de vos données](https://help.mistral.ai/en/articles/455207-can-i-opt-out-of-my-input-or-output-data-being-used-for-training) à tout moment.
- **Plans Team et Enterprise** : par défaut, les messages vocaux ne sont pas utilisés pour l’entraînement des modèles. Votre organisation doit explicitement donner son accord pour cela.

<SectionTab as="h2" sectionId="troubleshooting">Dépannage</SectionTab>

Si votre message vocal n’a pas été correctement transcrit :

- **Articulez clairement** et réduisez le bruit ambiant.
- **Vérifiez les autorisations micro** dans les paramètres du site de votre navigateur ou dans les réglages de l’application sur votre appareil.
- **Relisez et modifiez** la transcription dans la zone de saisie avant d’envoyer — surtout pour les noms propres, liens, chemins de fichiers ou chiffres.
