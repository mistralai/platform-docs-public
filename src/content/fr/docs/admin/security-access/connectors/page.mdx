# Gérer les Connecteurs

Les administrateurs de l'organisation gèrent les Connecteurs depuis la console Admin. Utilisez cette page pour connecter les bots de l'organisation, consulter les détails des Connecteurs et contrôler quels outils de Connecteur sont disponibles pour les utilisateurs dans Vibe et Studio.

:::info
L'administration des Connecteurs migre vers la console Admin. La configuration des Connecteurs, qui se faisait auparavant dans Vibe, sera gérée depuis Admin.
:::

<SectionTab as="h1" sectionId="open-connectors">Ouvrir les Connecteurs dans Admin</SectionTab>

1. Ouvrez <AppLink href="https://admin.mistral.ai/organization/connectors" app="admin">Connecteurs</AppLink> dans la console Admin.
2. Dans la section Organisation, cliquez sur `Connecteurs`.

La page comprend deux zones principales :

- `Connexions d'applications` : connectez des bots au niveau de l'organisation.
- `Tous les Connecteurs` : gérez les Connecteurs et leurs outils pour Vibe et Studio.

<SectionTab as="h1" sectionId="app-connections">Connecter des bots au niveau de l'organisation</SectionTab>

Utilisez `Connexions d'applications` pour connecter des services une seule fois au niveau de l'organisation. Une fois connectés, tous les membres peuvent les utiliser via Mistral sans avoir à se connecter individuellement.

| Application | Fonctionnalité |
|-----|--------------|
| `GitHub` | Rechercher des dépôts, fauire la relecture des problèmes et gérer les pull requests dans GitHub. |
| `Slack` | Rechercher des messages, lire des canaux, envoyer des messages et gérer des toiles dans Slack. |

Pour connecter une application, cliquez sur `Connecter à GitHub` ou `Connecter à Slack` et terminez le flux d'autorisation.

<SectionTab as="h1" sectionId="all-connectors">Gérer tous les Connecteurs</SectionTab>

`Tous les Connecteurs` répertorie les Connecteurs et outils pouvant être utilisés dans Vibe et Studio. Ouvrez un Connecteur pour consulter sa configuration et contrôler l'accès aux outils.

Chaque page de Connecteur includes trois onglets :

- `Aperçu` : consulter les métadonnées, la visibilité, l'authentification et le contenu disponible.
- `Identifiants` : consulter ou ajouter des identifiants pour le Connecteur.
- `Autorisations` : choisir quels outils peuvent être utilisés par le modèle.

<SectionTab as="h2" variant="secondary" sectionId="overview-tab">Consulter les détails du Connecteur</SectionTab>

L'onglet `Aperçu` affiche des informations de base sur le Connecteur, notamment :

| Champ | Exemple |
|-------|---------|
| `Visibilité` | `Organisation` |
| `Authentification` | `OAuth2` |
| `Origine` | `Recommandé` |
| `Capacités` | Aucune affichée |
| `Nom du Connecteur` | `atlassian` |
| `ID du Connecteur` | `0198e70f-57b0-77f6-a752-0a7f5ea2da35` |

Si vous n'êtes pas connecté, la section `Inclus` affiche `Vous n'êtes pas connecté`. Cliquez sur `Connecter` pour vous authentifier et voir les outils.

<SectionTab as="h2" variant="secondary" sectionId="credentials-tab">Gérer les identifiants</SectionTab>

L'onglet `Identifiants` répertorie les identifiants disponibles pour le Connecteur. S'il n'y a encore aucun identifiant, cliquez sur `Connecter` pour commencer à utiliser le Connecteur.

:::tip
Utilisez les connexions d'applications au niveau de l'organisation pour un accès partagé aux bots. Utilisez les identifiants du Connecteur lorsque les utilisateurs ou espaces de travail ont besoin d'une connexion de compte spécifique.
:::

<SectionTab as="h2" variant="secondary" sectionId="permissions-tab">Configurer les autorisations du Connecteur</SectionTab>

L'onglet `Autorisations` contrôle quels outils peuvent être utilisés par le modèle.

| Autorisation | Effet |
|------------|--------|
| `Autorisé` | Tous les outils sont disponibles pour les utilisateurs dans cet espace de travail. |
| `Restreint` | Seuls les outils sélectionnés sont disponibles pour les utilisateurs dans cet espace de travail. |
| `Bloqué` | Le Connecteur est désactivé pour les utilisateurs dans cet espace de travail. |

Utilisez `Restreint` lorsqu'un Connecteur inclut des outils qui ne doivent pas être disponibles pour tous les workflows ou discussions. Utilisez `Bloqué` lorsqu'un Connecteur ne doit pas être utilisé dans un espace de travail.

<SectionTab as="h1" sectionId="admin-vs-studio">Configuration Admin ou Studio</SectionTab>

Configurez et gérez les Connecteurs dans la console Admin. Utilisez les [Connecteurs Studio](/studio-api/connectors) pour accéder aux documents des Connecteurs destinés aux développeurs, aux exemples d'API ou au Débogueur de Connecteurs.

Nous mettrons les documents à jour au fur et à mesure que l'administration des Connecteurs terminera sa migration de Vibe vers Admin.
