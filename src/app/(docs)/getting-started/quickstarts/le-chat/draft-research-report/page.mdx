# Draft a research report

Draft a short research report using three core tools in <AppLink href="https://chat.mistral.ai">Le Chat</AppLink>.

- **Web Search**: pull in current data from the internet
- **Document Analysis**: upload a file and extract key points
- **Canvas**: refine the output in a side-by-side editor

By the end you'll have a formatted report and hands-on experience combining these tools in a single conversation.

**Time to complete:** ~10 minutes

<SectionTab as="h2" sectionId="prerequisites">Prerequisites</SectionTab>

- A Mistral AI account. The Free plan supports Web Search and Document Uploads; no subscription needed. <AppLink href="https://chat.mistral.ai">Create account</AppLink>

- Optional: a document to analyze (PDF, Word, or text format).

<SectionTab as="h2" sectionId="step-1">Step 1: Ask your first question with Web Search</SectionTab>

Le Chat can browse the web to find up-to-date information.

1. Open <AppLink href="https://chat.mistral.ai" app="le-chat">New chat</AppLink>.
2. Select **Web Search** icon (globe) under the input bar.
3. Ask a question about a recent event:

> What are the three most significant AI industry news announcements from the past 7 days? Summarize each in one sentence.

<SectionTab as="h2" sectionId="step-2">Step 2: Analyze a document</SectionTab>

Upload a document and let Le Chat extract the key points.

1. In the same chat, click the **paperclip icon** to upload a document (or provide a URL if you don't have one).
2. Ask Le Chat to extract insights. For example:

> Based on the context in this document, how does it relate to the news announcements you just found? Explain the connection step-by-step.

Le Chat reads the document and synthesizes the answer, keeping context from your previous search.

<SectionTab as="h2" sectionId="step-3">Step 3: Edit interactively with Canvas</SectionTab>

Canvas gives you a side-by-side workspace for editing long-form content with Mistral.

1. Ask Le Chat to generate a draft:

> Draft a short memo based on the document and the news events.

2. Click the **Open in Canvas** button near the response.
3. In the Canvas view on the right, highlight a sentence you want to change.
4. An inline prompt appears. Type "Make this sentence more professional" or "Expand on this idea".

<SectionTab as="h2" sectionId="verify">Verify</SectionTab>

You successfully used three of Le Chat's core capabilities:
1. **Web Search** for pulling in current data.
2. **Document Analysis** for extracting key points from uploaded files.
3. **Canvas** for interactive, side-by-side editing.
