# Langues prises en charge

Les modèles de langage de Mistral sont formés sur des données multilingues et prennent en charge un large éventail de langues. Les listes ci-dessous reflètent les langues pour lesquelles une **performance élevée** est attendue. En raison de leur entraînement, ces modèles peuvent également offrir de bons résultats dans d’autres langues non listées ici.

## Modèles de langage

Performance élevée attendue pour :

Arabe, persan, ourdou, hébreu, turc, indonésien, laotien, malais, thaï, tagalog, vietnamien, hindi, bengali, gujarati, kannada, marathi, népalais, pendjabi, tamoul, télougou, breton, catalan, tchèque, danois, grec, finnois, croate, néerlandais, norvégien, polonais, roumain, suédois, serbe, ukrainien, français, allemand, espagnol, portugais, italien, japonais, coréen, russe, chinois.

## OCR

Performance élevée attendue pour :

Anglais, allemand, français, russe, espagnol, néerlandais, italien, portugais, chinois, hongrois, coréen, turc, polonais, suédois, japonais, tchèque, danois, roumain, hébreu, tagalog, vietnamien, arabe, hindi, indonésien, marathi, norvégien, arménien, thaï, géorgien, gujarati, tamoul, bengali, persan, grec, finnois, ukrainien, télougou, népalais, pendjabi, kannada, catalan, serbe.

Les performances sont également bonnes pour des langues supplémentaires comme l’islandais, le malayalam, l’ourdou et le kazakh, entre autres.
