# Le Chat

Get hands-on with built-in tools in Le Chat. No code or API keys required.

<CtaButton href="https://chat.mistral.ai">Open Le Chat</CtaButton>

- **Web Search**: pull in current data from the internet
- **Document Analysis**: upload and extract insights from files
- **Canvas**: edit long-form content side-by-side with Mistral
- **Code Interpreter**: run Python for charts, tables, and calculations
- **Custom agents**: build reusable AI workflows with instructions and tools

Start with the research report quickstart to try core features, or jump to data analysis or agent creation.
