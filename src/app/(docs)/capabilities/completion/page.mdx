# Chat Completions

The Chat Completions API lets you **generate text using large language models** (LLMs) trained to **follow instructions and respond to natural language prompts** in conversation turns.

:::info
A _prompt_ is the input you send to the model: a question, an instruction, or a few examples of the task you want it to perform. The model generates a response based on that prompt.
:::

Common use cases include:
- Chatbots
- Classification
- Data extraction
- Text summarization
- Code generation
- Question answering
