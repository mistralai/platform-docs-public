# Studio

<AppLink href="https://console.mistral.ai" app="studio">Studio</AppLink> is Mistral's developer platform for AI applications. Use the API to build conversational AI, autonomous agents, document intelligence, and RAG applications. Use the console to manage reusable building blocks, manage keys, test models, and monitor usage.

<SectionTab as="h1" sectionId="studio-capabilities">Studio capabilities</SectionTab>

Studio gives you programmatic access to Mistral models for text generation, agents, data processing, and more.

<SectionTab as="h1" sectionId="getting-started-api">Getting started with the API</SectionTab>

1. **Create an API key** in the console or through the [API keys](/admin/security-access/api-keys) page in Admin.
2. **Send your first request**: follow the [send your first API request](/getting-started/quickstarts/developer/first-api-request) quickstart to make a chat completion call in Python or TypeScript.

:::tip
If you'd like to explore the features in depth, we suggest starting with the [chat completions](/studio-api/conversations/chat-completion) guide and then exploring other features based on your requirements.
:::

<SectionTab as="h1" sectionId="getting-started-console">Getting started with the console</SectionTab>

Open <AppLink href="https://console.mistral.ai" app="studio">Studio</AppLink> and head to the **Playground** to test prompts, compare models, and experiment with parameters: no code needed. From there, you can manage your API keys, review usage, and set up agents.

:::info
Some features may vary depending on your plan. Check the [pricing page](https://mistral.ai/pricing) for details.
:::
