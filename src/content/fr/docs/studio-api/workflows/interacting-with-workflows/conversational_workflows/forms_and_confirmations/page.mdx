# Formulaires et confirmations

Les workflows conversationnels peuvent demander à l’utilisateur des saisies structurées : champs à saisir avec validation, options à choix unique ou multiple, téléchargements de fichiers, confirmations accepter/refuser. Utilisez `FormInput` pour les formulaires complets et `ConfirmationInput` / `AcceptDeclineConfirmation` pour des demandes à validation rapide.

<SectionTab as="h1" sectionId="structured-form-inputs">Champs de formulaire structurés</SectionTab>

Pour les workflows nécessitant des saisies structurées avec des champs typés, de la validation et un rendu d’UI personnalisé, utilisez `FormInput` au lieu de `ChatInput` :

<Tabs>
  <TabItem value="python" label="Python">

```python
from datetime import date, datetime

import mistralai.workflows as workflows
import mistralai.workflows.plugins.mistralai as workflows_mistralai
from mistralai.workflows.conversational import (
    FormInput,
    TextField,
    NumberField,
    DateField,
    DateTimeField,
    SingleChoice,
)


class ExpenseForm(FormInput):
    """Formulaire structuré pour la saisie d’une dépense."""

    description: str = TextField(description="Description de la dépense")
    amount: float = NumberField(
        description="Montant en USD",
        minimum=0,
        maximum=10000,
    )
    category: str = SingleChoice(
        options=[
            ("travel", "Voyage"),
            ("equipment", "Équipement"),
            ("software", "Logiciel"),
        ],
        description="Catégorie de dépense",
    )
    expense_date: date = DateField(description="Date de la dépense")
    due_date: datetime = DateTimeField(description="Date limite de remboursement")
    receipt_id: str = TextField(
        description="ID du reçu",
        pattern=r"^RCP-\d{6}$",
    )


@workflows.workflow.define(
    name="expense-submission-workflow",
    workflow_display_name="Saisie de dépense",
    workflow_description="Saisir une dépense via un formulaire structuré",
)
class ExpenseSubmissionWorkflow(workflows.InteractiveWorkflow):
    @workflows.workflow.entrypoint
    async def run(self) -> workflows_mistralai.ChatAssistantWorkflowOutput:
        expense = await self.wait_for_input(
            ExpenseForm,
            label="Soumettre une dépense",
        )

        result = f"""Dépense enregistrée :
- Description : {expense.description}
- Montant : ${expense.amount:.2f}
- Catégorie : {expense.category}
- Date : {expense.expense_date.isoformat()}
- Date limite : {expense.due_date.isoformat()}
- Reçu : {expense.receipt_id}"""

        return workflows_mistralai.ChatAssistantWorkflowOutput(
            content=[workflows_mistralai.TextOutput(text=result)]
        )
```

<SectionTab as="h2" variant="secondary" sectionId="field-types">Types de champs</SectionTab>

| Type de champ   | Description        | Propriétés                                                                                          |
| --------------- | ----------------- | --------------------------------------------------------------------------------------------------- |
| `TextField`     | Champ texte        | `description` ; `pattern` (regex facultatif) ; `prefilled_value`                                    |
| `NumberField`   | Champ numérique    | `description`, `minimum`, `maximum`, `exclusive_minimum`, `exclusive_maximum`, `prefilled_value`   |
| `DateTimeField` | Sélecteur date/heure | `description`, `prefilled_value` (chaîne datetime ISO 8601)                                        |
| `DateField`     | Sélecteur date     | `description`, `prefilled_value` (chaîne date ISO 8601)                                            |
| `SingleChoice`  | Liste déroulante/choix unique | `options` (liste de tuples ou de chaînes), `description`, `prefilled_value`                |
| `MultiChoice`   | Sélection multiple | `options` (liste de tuples ou chaînes), `description`, `prefilled_value`                           |
| `FileField`     | Téléchargement de fichier | `description`, `multiple` (`False` par défaut), `include_metadata` (`False` par défaut)        |

Tous les types de champs (sauf `FileField`) acceptent en option un paramètre `prefilled_value`. C’est uniquement un indice d’UI : la valeur est préremplie mais le champ n’est *pas* rendu optionnel. L’utilisateur doit toujours valider la valeur. Si elle est invalide (hors limites, non conforme au pattern, option inconnue), elle est ignorée sans message.

Par défaut, le workflow reçoit des URLs (chaînes) des fichiers téléchargés par l’utilisateur. Avec `include_metadata=True`, il reçoit alors des objets `FileWithMetadataValue` :

| Champ          | Type        | Obligatoire | Description                                  |
| -------------- | ---------- | ----------- | --------------------------------------------- |
| `filename`     | `str`      | Oui         | Nom original du fichier                       |
| `url`          | `str`      | Oui         | URL signée pour télécharger le fichier        |
| `content_type` | `str`      | Oui         | Type MIME du fichier                         |

Ces URLs peuvent expirer : si votre workflow a besoin d’un accès prolongé, il doit stocker les fichiers ailleurs.

<SectionTab as="h1" sectionId="confirmation-inputs">Champs de confirmation</SectionTab>

Pour les workflows voulant une confirmation rapide sous forme de choix unique, utilisez `ConfirmationInput` ou `AcceptDeclineConfirmation`. Ces utilitaires créent un mini-formulaire où la sélection valide instantanément la réponse.

<SectionTab as="h2" variant="secondary" sectionId="confirmationinput">ConfirmationInput</SectionTab>

`ConfirmationInput` affiche une liste d’options sous forme de boutons. Le choix d’une option soumet immédiatement le formulaire :

<Tabs>
  <TabItem value="python" label="Python">

```python
import mistralai.workflows as workflows
import mistralai.workflows.plugins.mistralai as workflows_mistralai


@workflows.workflow.define(
    name="type-selection-workflow",
    workflow_display_name="Sélection du type",
    workflow_description="Choisissez votre type préféré",
)
class TypeSelectionWorkflow(workflows.InteractiveWorkflow):
    @workflows.workflow.entrypoint
    async def run(self) -> workflows_mistralai.ChatAssistantWorkflowOutput:
        await workflows_mistralai.send_assistant_message("Découvrons votre type préféré !")

        selection = await self.wait_for_input(
            workflows_mistralai.ConfirmationInput(
                options=[
                    ("fire", "Feu"),
                    ("water", "Eau"),
                    ("grass", "Plante"),
                    ("electric", "Électrik"),
                ],
                description="Quel est votre type préféré ?",
            )
        )

        selected_type = selection.choice  # Renvoie la valeur, ex : "fire"
        return workflows_mistralai.ChatAssistantWorkflowOutput(
            content=[workflows_mistralai.TextOutput(text=f"Type sélectionné : {selected_type} !")]
        )
```

<SectionTab as="h2" variant="secondary" sectionId="acceptdeclineconfirmation">AcceptDeclineConfirmation</SectionTab>

`AcceptDeclineConfirmation` est une confirmation spécialisée à deux options : accepter ou refuser. Les clients peuvent l’afficher comme standard, avec raccourcis clavier pour répondre rapidement :

<Tabs>
  <TabItem value="python" label="Python">

```python
import mistralai.workflows as workflows
import mistralai.workflows.plugins.mistralai as workflows_mistralai


@workflows.workflow.define(
    name="approval-workflow",
    workflow_display_name="Validation",
    workflow_description="Confirmer une action",
)
class ApprovalWorkflow(workflows.InteractiveWorkflow):
    @workflows.workflow.entrypoint
    async def run(self) -> workflows_mistralai.ChatAssistantWorkflowOutput:
        confirmation = await self.wait_for_input(
            workflows_mistralai.AcceptDeclineConfirmation(
                description="Voulez-vous poursuivre cette action ?",
                accept_label="Oui, poursuivre",
                decline_label="Annuler",
            )
        )

        if workflows_mistralai.is_accepted(confirmation):
            return workflows_mistralai.ChatAssistantWorkflowOutput(
                content=[workflows_mistralai.TextOutput(text="Action confirmée !")]
            )
        else:
            return workflows_mistralai.ChatAssistantWorkflowOutput(
                content=[workflows_mistralai.TextOutput(text="Action annulée.")]
            )
```
