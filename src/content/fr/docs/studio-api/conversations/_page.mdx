# Core API

Interagissez avec les modèles Mistral via une API unifiée pour la génération de texte, la compréhension d'images et l'extraction de données structurées. Que vous ayez besoin de simples chat completions ou de workflows multi-modaux avancés, ces endpoints constituent la base pour construire des applications propulsées par l'IA.

## Capacités

- [**Chat completion**](/studio-api/conversations/chat-completion) : Générez des réponses textuelles à partir de prompts en langage naturel via l'endpoint `/v1/chat/completions`. Prend en charge le streaming, les tokens d'arrêt, le prompting sécurisé et la génération basée sur des préfixes.

- [**Vision**](/studio-api/conversations/vision) : Traitez et comprenez des images en plus du texte. Comparez des images, transcrivez des documents, extrayez des données de graphiques et effectuez de l'OCR sur des documents historiques.

- [**Raisonnement**](/studio-api/conversations/reasoning) : Utilisez des modèles de raisonnement qui réfléchissent étape par étape avant de répondre, améliorant ainsi la précision sur des tâches complexes comme les mathématiques et le code.

- [**Citations**](/studio-api/conversations/citations) : Obtenez des références sources dans les réponses du modèle, permettant des sorties IA vérifiables et traçables, ancrées dans le contexte que vous fournissez.

- [**Sortie Structurée**](/studio-api/conversations/structured-output) : Contraignez les sorties du modèle à des schémas JSON valides ou utilisez le mode JSON pour obtenir des réponses prévisibles et lisibles par machine qui s'intègrent directement dans votre logique applicative.

- [**Appel de Fonctions**](/studio-api/conversations/function-calling) : Connectez les modèles à des outils et API externes en définissant des schémas JSON pour vos fonctions, permettant la récupération dynamique de données et l'exécution d'actions.

### Avancé

- [**Sorties prédites**](/studio-api/conversations/advanced/predicted-outputs) : Accélérez les réponses lorsque vous pouvez anticiper une partie de la sortie du modèle, ce qui réduit la latence pour les tâches de type édition.

- [**Mise en cache des prompts**](/studio-api/conversations/advanced/prompt-caching) : Réutilisez les jetons de prompt mis en cache pour les requêtes partageant des préfixes longs, comme dans les conversations multi-tours ou pour des prompts système répétés.
