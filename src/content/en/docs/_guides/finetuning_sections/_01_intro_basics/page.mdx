## Fine-tuning basics

Fine-tuning customizes a large language model (LLM) for a specific use case. By training a pretrained LLM on a labeled dataset for a particular task, fine-tuning can improve model performance. You can fine-tune a large model for complex or dissimilar tasks, or fine-tune a smaller model to approach the performance of a larger model with lower latency and cost. Results vary by use case.

Mistral provides a fine-tuning API in [Studio](https://console.mistral.ai/) for open-weight and commercial models. For experiments that need lower-level control, use the open-source [mistral-finetune](https://github.com/mistralai/mistral-finetune/) codebase. We recommend the API for most fine-tuning work because it doesn't require GPUs, supports multiple configuration options, and supports commercial models.
