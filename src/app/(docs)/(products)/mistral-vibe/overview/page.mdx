# Mistral Vibe

<AppLink href="https://console.mistral.ai/codestral/cli?utm_source=docs&utm_medium=header_cta&utm_campaign=studio_trial">Mistral Vibe</AppLink> is our open-source, terminal-native coding agent powered by Devstral. It scans your file structure and Git state, maintains multi-file context, and executes shell commands autonomously.

Whether you're refactoring across files, generating tests, or exploring an unfamiliar codebase, Mistral Vibe works right in your terminal. It also integrates with JetBrains and Zed via the Agent Communication Protocol, and can run fully offline with locally deployed models. This page covers what's available and how to get started.

<SectionTab as="h1" sectionId="capabilities">Capabilities</SectionTab>

Mistral Vibe brings together a conversational CLI, code generation, custom agents, offline support, and IDE integration.

<ExplorerTabs>
<ExplorerTab value="terminal" label="Terminal and CLI" default>

A conversational interface to your codebase, right in the terminal.

- **[CLI introduction](/mistral-vibe/terminal)**: install, configure, and start using Mistral Vibe. Interactive chat, `@` file references, `!` shell commands, slash-command skills, and persistent history.
- **[Installation](/mistral-vibe/terminal/install)**: step-by-step setup as a Python package with a quick installation guide.
- **[Configuration](/mistral-vibe/terminal/configuration)**: customize models, API keys, and behavior.
- **[Quickstart](/mistral-vibe/terminal/quickstart)**: get up and running with your first conversation.

</ExplorerTab>
<ExplorerTab value="coding" label="Coding">

Code completion and generation across 80+ languages.

- **[Coding](/mistral-vibe/using-fim-api)**: fill-in-the-middle completions and code generation with Codestral and Devstral models. Supports both completion and instruction-following modes.

</ExplorerTab>
<ExplorerTab value="agents-skills" label="Agents and skills">

Build custom agents for targeted tasks and extend them with skills.

- **[Agents and skills](/mistral-vibe/agents-skills)**: create custom subagents for specific use cases (deploy scripts, PR reviews, test generation), add slash-command skills, and set up multi-choice clarification workflows.

</ExplorerTab>
<ExplorerTab value="offline-local" label="Offline and local">

Run without internet on your own infrastructure.

- **[Offline / local](/mistral-vibe/local)**: deploy Devstral locally and use Mistral Vibe without an internet connection. Devstral Small supports consumer GPUs (GeForce RTX), NVIDIA DGX Spark, and CPU-only configurations.

</ExplorerTab>
</ExplorerTabs>
