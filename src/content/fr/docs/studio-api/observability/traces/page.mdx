# Traçage distribué

Chaque requête dans votre application d'IA est capturée sous forme de **trace** : un arbre de **spans** représentant chaque étape de la chaîne d'exécution. Chaque span contient ses entrées, sorties, latence, compte de token et statut.

Les traces sont collectées à l'aide de [OpenTelemetry](https://opentelemetry.io/) et sont prises en charge dans l'ensemble des produits Mistral et le SDK Mistral.

:::info
**Conservation des données** : les traces sont conservées pendant **30 jours**.
:::
