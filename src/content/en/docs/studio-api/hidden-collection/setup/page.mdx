# Hidden collection setup

This page checks that child pages under a hidden category stay available from direct URLs.

## Configure the test

1. Start the local docs server.
2. Open `/studio-api/hidden-collection/setup` directly.
3. Confirm the page renders and doesn't appear in the sidebar.

## Expected result

The page should load from the direct URL and remain absent from navigation surfaces.
