# Installer et s’authentifier

Installez **[Mistral Vibe pour VS Code](https://marketplace.visualstudio.com/items?itemName=mistralai.mistral-vibe-code)**, ouvrez le panneau Vibe, puis connectez-vous. L’extension inclut l’agent Vibe, **aucune installation séparée du CLI n’est requise**.

:::tip
**Switching surface?** This page covers the **VS Code extension**. To set up Vibe in your terminal, see [Installer le CLI](/vibe/code/cli/install-setup); for a cloud session on a GitHub repo, see [Vibe Code Web](/vibe/code/vibe-code-web/get-started). Configuration and sessions are shared between the CLI and the extension.
:::

<SectionTab as="h2" sectionId="requirements">Conditions préalables</SectionTab>

| Condition préalable | Détails |
|---|---|
| **VS Code** | Version 1.94.0 ou ultérieure. |
| **Compte Mistral** | Requis pour créer ou gérer les clés API. |
| **Accès au service** | Un [plan Mistral](https://mistral.ai/pricing) (Gratuit, Pro ou supérieur) ou un plan API avec une clé que vous gérez vous-même. Des limites d’usage et de débit s’appliquent. |

<SectionTab as="h2" sectionId="install">Installer l’extension</SectionTab>

Installez **Mistral Vibe pour VS Code** depuis la vue Extensions de VS Code ou directement depuis la [page Marketplace](https://marketplace.visualstudio.com/items?itemName=mistralai.mistral-vibe-code).

Depuis VS Code :

1. Ouvrez VS Code.
2. Appuyez sur `Cmd+Maj+X` sur macOS ou `Ctrl+Maj+X` sur Windows ou Linux.
3. Recherchez `Mistral Vibe`.
4. Sélectionnez l’extension publiée par **Mistral AI** (ID de l’éditeur : `mistralai`).
5. Cliquez sur `Installer`.

Si l’extension n’apparaît pas après l’installation, exécutez **Developer : Reload Window** depuis la palette de commandes ou redémarrez VS Code.

<SectionTab as="h2" sectionId="open">Ouvrir Mistral Vibe</SectionTab>

Une fois l’installation terminée, ouvrez l’extension depuis l’un de ces points d’entrée :

- Cliquez sur l’icône **Mistral Vibe** dans la barre d’activité de VS Code.
- Exécutez **Mistral Vibe : Open Mistral Vibe** depuis la palette de commandes.
- Utilisez l’action du titre de l’éditeur lorsque l’icône Mistral Vibe est disponible pour le fichier actif.

L’extension ouvre un panneau de conversation web dans VS Code.

<SectionTab as="h2" sectionId="authenticate">S’authentifier</SectionTab>

La première fois que l’extension a besoin d’identifiants, elle ouvre un **flux de configuration dans le terminal** :

1. Lancez la configuration depuis l’extension à l’invite.
2. Dans le terminal **Mistral Vibe Setup**, collez votre clé API Mistral.
3. Terminez l’invite et attendez que l’extension revienne à l’état prêt.

Le flux de configuration exécute l’agent Vibe intégré avec `--setup`. Il stocke les identifiants dans le répertoire d’accueil de Vibe pour les sessions ultérieures.

:::info
**La connexion via navigateur est en cours d’activation pour l’agent Vibe intégré.** Lorsque votre configuration cible un fournisseur Mistral, vous serez connecté via un flux navigateur au lieu de coller une clé API. En attendant qu’elle soit disponible dans l’extension, utilisez le flux de configuration dans le terminal ci-dessus.
:::

<SectionTab as="h2" sectionId="verify">Vérifier la configuration</SectionTab>

Votre configuration est prête lorsque :

- Le panneau Mistral Vibe s’ouvre sans invite d’authentification.
- Vous pouvez envoyer un prompt depuis le panneau VS Code.
- L’extension peut utiliser l’espace de travail et le contexte de l’éditeur actifs.

Essayez un premier prompt simple :

```text
Explain the currently open file.
```

<SectionTab as="h2" sectionId="vsix">Installer depuis un VSIX</SectionTab>

Si votre équipe reçoit une version `.vsix` avant la publication publique, installez-la depuis **Extensions > Autres actions > Installer depuis VSIX...**.

:::warning
Utilisez uniquement les fichiers VSIX provenant d’un **canal de distribution Mistral approuvé**.
:::
