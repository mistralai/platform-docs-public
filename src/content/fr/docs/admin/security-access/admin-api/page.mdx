# Admin API

L'Admin API fournit un accès programmatique aux tâches de gestion d'organisation autrement disponibles via le <AppLink href="https://admin.mistral.ai" app="admin">Panneau d'administration</AppLink>.

<SectionTab as="h1" sectionId="capabilities">Fonctionnalités</SectionTab>

Utilisez l'Admin API pour automatiser :

- La gestion de l'organisation et des espaces de travail
- Les invitations d'utilisateurs et les attributions de rôles
- Le provisionnement des sièges et le contrôle d'accès
- Les requêtes de facturation et d'utilisation

<SectionTab as="h1" sectionId="authentication">Authentification</SectionTab>

Les requêtes Admin API nécessitent une clé API appartenant à un utilisateur ayant le rôle **Admin** dans l'organisation. Les clés API standard avec les rôles Member ou Billing n'ont pas accès aux endpoints administratifs.

<SectionTab as="h1" sectionId="reference">Référence API</SectionTab>

Consultez la [référence API](/api) pour la documentation des endpoints.
