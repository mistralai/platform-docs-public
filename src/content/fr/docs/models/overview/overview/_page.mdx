## Dépréciation des modèles

### Vue d'ensemble
Notre offre de modèles est continuellement renouvelée avec des modèles plus récents et plus performants. Dans ce cadre, nous déprécions et retirons les modèles plus anciens. Ce document fournit des informations sur les modèles actuellement disponibles, dépréciés ou retirés.

### Terminologie
- Date de dépréciation : la date à laquelle le modèle est marqué comme déprécié. Lorsqu'un modèle est déprécié, il reste disponible pour les clients ayant des déploiements existants jusqu'à ce que le modèle soit retiré. 
- Date de retrait : la date à laquelle le modèle est marqué comme retiré. Lorsqu'un modèle est retiré de Studio, il n'est plus disponible et, lorsqu'il est sollicité, il renvoie une réponse d'erreur.

### Comment se préparer aux retraits de modèles et aux mises à jour de version
Pour se préparer aux retraits de modèles et aux mises à jour de version, nous recommandons aux clients d'évaluer leurs applications avec les nouveaux modèles et versions, et d'analyser leur comportement. Nous recommandons également aux clients de mettre à jour leurs applications pour utiliser les nouveaux modèles et versions avant la date de retrait
