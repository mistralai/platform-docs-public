# Document AI

Mistral Document AI propose un traitement documentaire de niveau entreprise, associant la technologie OCR à une extraction avancée de données structurées.

<Image
  url={['/img/document_ai_overview.png', '/img/document_ai_overview_dark.png']}
  alt="document_ai_graph"
  width="500px"
  centered
/>

Bénéficiez de la prise en charge multilingue, d’annotations et de workflows adaptables pour tous les types de documents afin d’extraire, comprendre et analyser les données.

<SectionTab as="h1" sectionId="document-ai-services">Services Document AI</SectionTab>

Utilisez `client.ocr.process` dans nos SDK clients ou l’endpoint `https://api.mistral.ai/v1/ocr` pour accéder aux services suivants :

- [OCR Processor](/studio-api/document-processing/basic_ocr) : Découvrez notre modèle OCR ainsi que ses fonctionnalités.
- [Annotations](/studio-api/document-processing/annotations) : Annotez et extrayez des données de vos documents grâce aux sorties structurées intégrées.
- [Document QnA](/studio-api/document-processing/document_qna) : Combinez nos modèles avec la technologie OCR pour l’extraction de réponses dans les documents.
