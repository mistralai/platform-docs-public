# RAG et embeddings

Créez des pipelines de génération augmentée par retrieval personnalisés avec des embeddings, une recherche vectorielle et la génération de modèle. Ces guides expliquent le fonctionnement de la retrieval sous le capot et comment construire un pipeline RAG à partir de zéro.

Si vous souhaitez une fonctionnalité gérée qui ingère, vectorise et recherche des documents pour vous, utilisez [Bibliothèques](/studio-api/libraries). Si vous souhaitez ancrer les réponses dans des sources connectées comme Google Drive ou SharePoint, utilisez [Connecteurs](/studio-api/connectors).
